NAME
  terminalSize - report the current size of the terminal TUI surface

SYNOPSIS
  term::terminalSize() AS TermSize

PACKAGE
  term

IMPORTS
  IMPORT term

DESCRIPTION
  Returns the current size of the term:: surface as a TermSize record holding two
  Integer fields: columns, the number of character cells across the surface
  (its width), and rows, the number of character cells down the surface (its
  height). The dimensions are measured in whole character cells, not pixels, and
  describe the drawable grid the rest of the term:: surface writes into: valid
  cursor positions span rows 0 through rows-1 and columns 0 through columns-1.

  The reported size reflects the backend the program is running on. On the
  console backend it is the size of the controlling terminal window at the time
  of the call, so resizing the terminal and calling term::terminalSize again can
  return different values. In windowed app mode (mfb build -app) it is the size
  of the surface grid derived from the current window dimensions. Because the
  surface can be resized between calls, a program that relies on the dimensions
  (for layout, centering, or bounds checks) should call term::terminalSize again
  after a possible resize rather than caching the first result.

  Like the rest of the term:: surface other than term::on and term::isOn,
  term::terminalSize is gated behind TUI mode: it is a no-op while TUI mode is
  off, so a program must call term::on before the returned size is meaningful.
  The function takes no arguments and has no side effects: it only reads the
  current surface size and does not change any term:: state, the cursor, or the
  screen.

PARAMETERS
  None.

RETURN VALUE
  AS TermSize
    Returns a TermSize record whose columns field is the width of the surface in
    character cells and whose rows field is the height of the surface in
    character cells. Both are positive Integer counts; valid cursor coordinates
    run from 0 to columns-1 horizontally and 0 to rows-1 vertically.

ERRORS
  No errors.

EXAMPLES
  Read the surface dimensions and print them:

    IMPORT term
    IMPORT io

    term::on()
    LET size AS TermSize = term::terminalSize()
    io::print(toString(size.columns))
    io::print(toString(size.rows))
    term::off()

  Move the cursor to the center of the surface:

    IMPORT term
    IMPORT io

    term::on()
    LET size AS TermSize = term::terminalSize()
    term::moveTo(size.rows / 2, size.columns / 2)
    io::print("middle")
    term::off()
