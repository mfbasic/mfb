NAME
  contains - test whether a list contains an item

SYNOPSIS
  collections::contains OF T(value AS List OF T, item AS T) AS Boolean

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  contains scans value from left to right and returns TRUE as soon as an element
  equal to item is found, or FALSE when no element matches. The list is not
  copied or mutated.

OVERLOADS
  contains OF T(value AS List OF T, item AS T) AS Boolean
    Returns TRUE when item appears in value.

ERRORS
  contains does not fail for supported argument types. Errors can still propagate
  from evaluating arguments before contains is called.

TYPE CHECKING
  The first argument must be a List OF T. The second argument must be an item of
  the list element type T. Calling contains on a non-list value, or with a
  mismatched element type, is a compile-time type error.

EXAMPLES
  Check for a value:

    IF contains([1, 2, 3], 2) THEN
      RETURN 0
    END IF

SEE ALSO
  find, filter, len
