NAME
    window - produce sliding windows over a list

SYNOPSIS
    collections::window OF T(value AS List OF T, size AS Integer, stride AS Integer = 1) AS List OF List OF T

DESCRIPTION
    Returns a list of overlapping windows over value. Each window is a list of
    size consecutive items, and successive windows advance by stride positions.
    stride defaults to 1. When size is greater than the length of value, no
    window is produced and the result is empty. value is not modified.

EXAMPLES
    IMPORT io
    IMPORT collections

    FUNC main AS Integer
      LET windows AS List OF List OF Integer = collections::window([1, 2, 3, 4], 2)
      io::print(toString(len(windows)))
      RETURN 0
    END FUNC

ERRORS
    77050002 (ErrInvalidArgument) when size is less than 1 or stride is less than 1.
