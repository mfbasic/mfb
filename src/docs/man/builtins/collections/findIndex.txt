NAME
    findIndex - index of the first item satisfying a predicate

SYNOPSIS
    collections::findIndex OF T(value AS List OF T, predicate AS FUNC(T) AS Boolean, start AS Integer = 0) AS Integer

DESCRIPTION
    Returns the zero-based index of the first item at or after start for which
    predicate returns TRUE. The search begins at start, which defaults to 0.
    Pass predicate as a named FUNC value.

EXAMPLES
    IMPORT io
    IMPORT collections

    FUNC isPos(n AS Integer) AS Boolean
      RETURN n > 0
    END FUNC

    FUNC main AS Integer
      io::print(toString(collections::findIndex([-1, 0, 3, 4], isPos)))
      RETURN 0
    END FUNC

ERRORS
    77050004 (ErrNotFound) when no item at or after start satisfies predicate.
    77050001 (ErrIndexOutOfRange) when start is out of range.
