NAME
  filter - keep list items that pass a predicate

SYNOPSIS
  collections::filter OF T(value AS List OF T, predicate AS FUNC(T) AS Boolean) AS List OF T

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  filter calls predicate once for each item in value, in order from left to
  right, and returns a new list containing the items for which predicate
  returns TRUE. The order of the kept items is preserved. The original list is
  neither copied for predicate nor mutated.

  Built-in predicate helpers such as isEven, isOdd, isPositive, isNegative,
  isZero, isEmpty, and isNotEmpty are intended for direct use with filter when
  their argument type matches the list element type.

  If predicate fails for any item, the error auto-propagates and iteration
  stops, so later items are not visited and no result list is returned.

OVERLOADS
  filter OF T(value AS List OF T, predicate AS FUNC(T) AS Boolean) AS List OF T
    Keeps items where predicate returns TRUE and returns a List OF T.

ERRORS
  filter propagates errors raised by predicate, stopping at the failing item.
  Errors can still propagate from evaluating value or predicate before filter
  is called. An inline TRAP attached to a filter call captures that failure at
  the call site instead of auto-propagating (the partially-built result is freed
  first).

TYPE CHECKING
  The first argument must be a List OF T. The second argument must accept one
  argument of the list element type T and return Boolean. The result list
  element type is the same T as the input. Calling filter on a non-list value,
  with a predicate whose parameter type does not match T, or with a predicate
  that does not return Boolean, is a compile-time type error.

EXAMPLES
  Keep even numbers:

    LET evens AS List OF Integer = filter([1, 2, 3, 4], isEven)

  Keep non-empty strings:

    LET names AS List OF String = filter(["Ada", "", "Grace"], isNotEmpty)

  Use a local function:

    FUNC isLong(word AS String) AS Boolean
      RETURN len(word) > 3
    END FUNC

    LET long AS List OF String = filter(["hi", "hello", "ok"], isLong)
