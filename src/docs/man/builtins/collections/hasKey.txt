NAME
  hasKey - test whether a map contains a key

SYNOPSIS
  collections::hasKey OF K, V(value AS Map OF K TO V, key AS K) AS Boolean

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  hasKey returns TRUE when value contains an entry for key and FALSE otherwise.
  The map is not copied or mutated.

  Use hasKey to test for membership before reading with get, which fails on a
  missing key. When you only need a value with a fallback, getOr avoids the
  separate membership test.

OVERLOADS
  hasKey OF K, V(value AS Map OF K TO V, key AS K) AS Boolean
    Tests for key membership.

ERRORS
  hasKey does not fail for missing keys. Errors can still propagate from
  evaluating arguments before hasKey is called.

TYPE CHECKING
  key must have the map key type K.

EXAMPLES
  Test map membership:

    LET ok AS Boolean = hasKey(Map OF String TO Integer { "a" := 1 }, "a")
