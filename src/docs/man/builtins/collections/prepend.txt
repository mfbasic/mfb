NAME
  prepend - return a list with an item added at the start

SYNOPSIS
  collections::prepend OF T(value AS List OF T, item AS T) AS List OF T

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  prepend returns a new list with item inserted before the existing first item.
  The original list is not mutated.

  The result is one element longer than value, with item placed ahead of every
  existing element in their original order.

OVERLOADS
  prepend OF T(value AS List OF T, item AS T) AS List OF T
    Returns a list with item added at the start.

ERRORS
  prepend does not fail for supported argument types. Errors can still propagate
  from evaluating arguments before prepend is called.

TYPE CHECKING
  The first argument must be a List OF T. The second argument must be an item of
  the list element type T. Calling prepend on a non-list value, or with a
  mismatched element type, is a compile-time type error.

EXAMPLES
  Add an item to the start:

    LET numbers AS List OF Integer = prepend([2, 3], 1)

SEE ALSO
  append, insert, set, removeAt, len
