NAME
    chunks - split a list into consecutive fixed-size chunks

SYNOPSIS
    collections::chunks OF T(value AS List OF T, chunkSize AS Integer) AS List OF List OF T

DESCRIPTION
    Returns a list of consecutive, non-overlapping chunks of value, each of
    length chunkSize. The final chunk may be shorter when the length of value
    is not a multiple of chunkSize. An empty input list yields an empty result.
    value is not modified.

EXAMPLES
    IMPORT io
    IMPORT collections

    FUNC main AS Integer
      LET parts AS List OF List OF Integer = collections::chunks([1, 2, 3, 4, 5], 2)
      io::print(toString(len(parts)))
      RETURN 0
    END FUNC

ERRORS
    77050002 (ErrInvalidArgument) when chunkSize is less than 1.
