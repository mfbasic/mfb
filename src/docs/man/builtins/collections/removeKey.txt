NAME
  removeKey - remove one map entry by key

SYNOPSIS
  collections::removeKey OF K, V(value AS Map OF K TO V, key AS K) AS Map OF K TO V

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  removeKey returns a new map with the entry for key removed. The original map is
  not mutated.

  Missing keys are ignored, so removing a key that is not present returns a map
  equivalent to the input map. The returned map is the same size as value, or one
  entry shorter when key was present.

OVERLOADS
  removeKey OF K, V(value AS Map OF K TO V, key AS K) AS Map OF K TO V
    Returns a map without key.

ERRORS
  removeKey does not fail when key is absent. Errors can still propagate from
  evaluating arguments before removeKey is called.

TYPE CHECKING
  value must be a Map OF K TO V and key must have the map key type K. The result has
  the same map type as value.

EXAMPLES
  Remove a key:

    LET scores AS Map OF String TO Integer = removeKey(Map OF String TO Integer { "a" := 1 }, "a")
