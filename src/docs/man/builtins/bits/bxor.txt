NAME
  bxor - bitwise exclusive-OR

SYNOPSIS
  bits::bxor(a AS Integer, b AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Returns the bitwise exclusive-OR of a and b, computed independently across all
  64 bits: bit i of the result is 1 when bit i differs between the two operands,
  and 0 when the two bits are equal.

  Operands are raw two's-complement 64-bit Integer bit patterns; the function does
  not interpret sign. The operation is total, has no side effects, and lowers to a
  single native AArch64 instruction inline, producing identical results on the
  native and Binary Representation execution paths.

PARAMETERS
  a AS Integer
    The first operand. Any 64-bit value.

  b AS Integer
    The second operand. Any 64-bit value.

RETURN VALUE
  AS Integer
    The bitwise XOR of a and b.

ERRORS
  No errors.

EXAMPLES
  Toggle the low byte of a value:

    IMPORT bits

    LET result AS Integer = bits::bxor(value, 255)
