NAME
  sra - arithmetic right shift

SYNOPSIS
  bits::sra(value AS Integer, count AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Shifts value right by count bits as a signed quantity; vacated high bits are filled with the sign bit (bit 63 of value).

  A count of 0 returns value unchanged. count must be in 0..63 (a 64-bit
  Integer); larger shifts are not implicitly clamped and instead fail.

  Operands are raw two's-complement bit patterns; the function does not interpret
  sign except as its signature states. Unlike bits::sr, the sign bit is
  replicated into the vacated high bits, so shifting a negative value keeps it
  negative.

PARAMETERS
  value AS Integer
    The value to shift.
  count AS Integer
    The shift amount, 0..63.

RETURN VALUE
  AS Integer
    value shifted right (arithmetic) by count.

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when count is outside 0..63.

EXAMPLES
  IMPORT bits

  LET result AS Integer = bits::sra(1, 4)
