NAME
  rl64 - rotate all 64 bits left

SYNOPSIS
  bits::rl64(value AS Integer, count AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Rotates all 64 bits of value left by count bit positions and returns the
  result. The rotate is a full-width 64-bit barrel rotate: bits shifted out of
  bit 63 re-enter at bit 0, so no information is lost.

  The rotate amount is reduced modulo 64, so any count produces a defined result
  and counts of 0 and 64 both leave value unchanged. Negative counts are also
  reduced modulo 64 (a left rotate by a negative amount is equivalent to a right
  rotate).

  Operands are raw two's-complement bit patterns; the function does not interpret
  sign except as its signature states.

PARAMETERS
  value AS Integer
    The value whose 64 bits are rotated.

  count AS Integer
    The number of bit positions to rotate left, reduced modulo 64.

RETURN VALUE
  AS Integer
    value rotated left by count MOD 64.

ERRORS
  No errors.

EXAMPLES
  Rotate all 64 bits left by four positions:

    IMPORT bits

    LET result AS Integer = bits::rl64(1, 4)
