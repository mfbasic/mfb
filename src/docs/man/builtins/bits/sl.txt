NAME
  sl - logical left shift

SYNOPSIS
  bits::sl(value AS Integer, count AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Shifts value left by count bits; vacated low bits are zero and bits shifted past bit 63 are discarded.

  Operands are raw two's-complement bit patterns; the function does not interpret
  sign except as its signature states.

PARAMETERS
  value AS Integer
    The value to shift.
  count AS Integer
    The shift amount, 0..63.

RETURN VALUE
  AS Integer
    value shifted left by count.

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when count is outside 0..63.

EXAMPLES
  IMPORT bits

  LET result AS Integer = bits::sl(1, 4)
