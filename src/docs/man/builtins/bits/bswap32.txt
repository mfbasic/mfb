NAME
  bswap32 - reverse the byte order of the low 32 bits

SYNOPSIS
  bits::bswap32(value AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Reverses the order of the four bytes that make up the low 32 bits of value and
  returns the result. Byte 0 becomes byte 3, byte 1 becomes byte 2, byte 2
  becomes byte 1, and byte 3 becomes byte 0. All bits above bit 31 (bits 32..63)
  are cleared to zero in the result.

  The operand is treated as a raw two's-complement bit pattern; the function does
  not interpret its sign except as the Integer signature states. The operation is
  total: it is defined for every Integer value and never fails.

PARAMETERS
  value AS Integer
    The value whose low 32 bits are byte-reversed. Bits above bit 31 are ignored
    and do not appear in the result.

RETURN VALUE
  AS Integer
    The byte-reversed low 32 bits of value, with bits 32..63 cleared to zero. The
    result is always in the range 0..4294967295.

ERRORS
  No errors.

EXAMPLES
  Swap the four low bytes of a 32-bit value:

    IMPORT bits

    LET result AS Integer = bits::bswap32(255)
