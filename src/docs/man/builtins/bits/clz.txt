NAME
  clz - count leading zeros

SYNOPSIS
  bits::clz(value AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Returns the number of zero bits that precede the most significant set (1)
  bit of the 64-bit value, counting from bit 63 (the highest bit) downward.
  The value is treated as a raw two's-complement 64-bit bit pattern; the sign
  is not interpreted. When value is 0 there is no set bit, so all 64 bits are
  leading zeros and the result is 64. This operation is total: it is defined
  for every Integer and never fails. It lowers to a single native AArch64
  count-leading-zeros instruction.

PARAMETERS
  value AS Integer
    The 64-bit value to inspect. Any Integer is accepted.

RETURN VALUE
  AS Integer
    The leading-zero count, in the range 0..64. The result is 0 when bit 63
    is set, and 64 when value is 0.

ERRORS
  No errors.

EXAMPLES
  Count the leading zeros of a small value:

    IMPORT bits

    LET result AS Integer = bits::clz(255)
