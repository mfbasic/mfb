NAME
  toFloat - convert text, Integer, or Fixed to Float

SYNOPSIS
  toFloat(value AS String) AS Float
  toFloat(value AS Integer) AS Float
  toFloat(value AS Fixed) AS Float

PACKAGE
  general

IMPORTS
  None. general functions are always available without an IMPORT statement.

DESCRIPTION
  toFloat converts a supported value to a 64-bit IEEE 754 binary64 Float.

  The String overload parses decimal Float text. An optional leading minus sign
  is accepted, followed by digits and an optional fractional part; the entire
  string must form a single well-formed decimal number, so surrounding
  whitespace and other stray characters are rejected. Parsing must yield a
  finite value: malformed text and text that names a non-finite value such as
  NaN or Infinity are rejected, and text whose magnitude is too large to
  represent and would round to infinity also fails.

  The Integer overload converts a signed 64-bit Integer to the nearest
  representable Float, rounding to nearest when the Integer has more significant
  bits than the Float mantissa can hold. Every Integer maps to a finite Float,
  so this conversion always succeeds.

  The Fixed overload converts a Fixed value to the nearest representable Float.
  Every Fixed value maps to a finite Float, so this conversion always succeeds.

OVERLOADS
  toFloat(value AS String) AS Float
    Parses a finite Float value from decimal text.

  toFloat(value AS Integer) AS Float
    Converts an Integer to the nearest representable Float.

  toFloat(value AS Fixed) AS Float
    Converts a Fixed value to the nearest representable Float.

PARAMETERS
  value AS String
    Text holding a decimal Float, optionally prefixed with a minus sign and
    including an optional fractional part. The whole string must be valid; no
    leading, trailing, or interior extra characters are permitted.

  value AS Integer
    A signed 64-bit value to convert to the nearest representable Float.

  value AS Fixed
    A Fixed value to convert to the nearest representable Float.

RETURN VALUE
  AS Float
    The Float equivalent of value. For String input the parsed finite Float is
    returned. For Integer and Fixed input the nearest representable Float is
    returned, which may differ slightly from the source value when it cannot be
    represented exactly.

ERRORS
  77050003 (ErrInvalidFormat)
  Raised when a String value is not well-formed decimal Float text, or names a
  non-finite value such as NaN or Infinity.

  77050010 (ErrOverflow)
  Raised when parsing a String value yields a magnitude too large to represent
  and would round to infinity.

TYPE CHECKING
  toFloat accepts only String, Integer, and Fixed values. Calling it with any
  other type is a compile-time error; convert such values explicitly first.

EXAMPLES
  Parse text:

    LET value AS Float = toFloat("1.5")

  Convert an Integer:

    LET value AS Float = toFloat(42)

  Convert a Fixed value:

    LET fixed AS Fixed = toFixed("2.25")
    LET value AS Float = toFloat(fixed)
