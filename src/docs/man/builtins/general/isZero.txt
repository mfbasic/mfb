NAME
  isZero - test whether a number is equal to zero

SYNOPSIS
  isZero(value AS Integer) AS Boolean
  isZero(value AS Float) AS Boolean
  isZero(value AS Fixed) AS Boolean

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  isZero returns TRUE when value is equal to zero and FALSE otherwise.

  The test is the equality test value = 0. Positive values are not zero, so
  isZero(5) is FALSE, and negative values are not zero, so isZero(-5) is FALSE.
  Only an exact zero yields TRUE.

  isZero is overloaded for the three numeric types: Integer, Float, and Fixed.
  The argument is compared against zero in its own type with no conversion, so
  the same rule applies whether value is 0, 0.0, or a Fixed amount. Negative
  zero, where the type permits it, compares equal to zero and therefore yields
  TRUE.

  isZero is useful in ordinary conditionals, MATCH guards, and as a predicate
  passed to filter.

OVERLOADS
  isZero(value AS Integer) AS Boolean
    Tests an Integer against zero, returning TRUE when value = 0.

  isZero(value AS Float) AS Boolean
    Tests a Float against zero, returning TRUE when value = 0.0.

  isZero(value AS Fixed) AS Boolean
    Tests a Fixed against zero, returning TRUE when value is equal to zero.

PARAMETERS
  value AS Integer
    The number to test. Any Integer, Float, or Fixed is accepted; its value
    determines the result.

RETURN VALUE
  AS Boolean
    TRUE when value is equal to zero, FALSE otherwise. Positive and negative
    values both return FALSE.

ERRORS
  No errors.

TYPE CHECKING
  isZero accepts a single Integer, Float, or Fixed value. Calling it with any
  other type, or with a different number of arguments, is a compile-time error.

EXAMPLES
  Test a literal:

    LET result AS Boolean = isZero(0)

  Branch on the value:

    IF isZero(balance) THEN
      PRINT "balance is zero"
    END IF

  Filter a list:

    LET zeros AS List OF Integer = filter([-1, 0, 2, 0], isZero)
