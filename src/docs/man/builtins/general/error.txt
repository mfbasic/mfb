NAME
  error - construct an Error value with a code and message

SYNOPSIS
  error(code AS Integer, message AS String) AS Error

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  error constructs an Error value from an Integer code and a String message. The
  resulting value is a read-only built-in record carrying the supplied code and
  message together with a compiler-generated source location captured at the call
  site. Neither argument is interpreted, validated, or constrained: any Integer
  code and any String message, including a zero code or an empty message, produce
  a valid Error.

  Error and ErrorLoc are read-only built-in records. Programs create errors with
  error and inspect their fields; the fields cannot be reassigned. Binding and
  propagating an Error preserve its fields unchanged. error has no side effects
  beyond producing the value.

  An Error value exposes the following fields:

    code     AS Integer   the code passed to error
    message  AS String    the message passed to error
    source   AS ErrorLoc  the call site where error was evaluated

  An ErrorLoc value exposes the following fields:

    filename AS String    the source file containing the error call
    line     AS Integer    the line of the error call
    char     AS Integer    the column of the error call

PARAMETERS
  code AS Integer
    The numeric error code stored in the Error's code field. Any Integer value is
    accepted and recorded verbatim, with no reserved or invalid values.

  message AS String
    The human-readable message stored in the Error's message field. Any String,
    including an empty one, is accepted and recorded verbatim.

RETURN VALUE
  AS Error
    A read-only Error record whose code and message fields equal the supplied
    arguments and whose source field holds the ErrorLoc of the call site.

ERRORS
  No errors.

EXAMPLES
  Construct an Error and inspect its fields:

    LET value AS Error = error(123, "boom")
    IF value.code <> 123 THEN RETURN 1
    IF value.message <> "boom" THEN RETURN 2

  Return an Error from a function:

    FUNC makeError(code AS Integer, message AS String) AS Error
      RETURN error(code, message)
    END FUNC

  Report the source location of an Error:

    LET value AS Error = error(500, "internal")
    LET where AS ErrorLoc = value.source
    io::print(where.filename & ":" & toString(where.line) & ":" & toString(where.char))
