NAME
  poll - test a stream for readiness

SYNOPSIS
  audio::poll(stream AS AudioInput) AS Boolean
  audio::poll(stream AS AudioOutput) AS Boolean
  audio::poll(stream AS AudioInput, timeoutMs AS Integer) AS Boolean
  audio::poll(stream AS AudioOutput, timeoutMs AS Integer) AS Boolean

PACKAGE
  audio

IMPORTS
  IMPORT audio

DESCRIPTION
  Reports whether a stream is ready without blocking. For an AudioInput, ready
  means at least one frame can be read; for an AudioOutput, at least one frame
  can be written. audio::poll(stream) is exactly audio::available(stream) > 0.

  The timed form waits up to timeoutMs milliseconds for the stream to become
  ready, returning TRUE as soon as it is or FALSE on timeout. A timeoutMs of 0
  is a non-blocking test.

OVERLOADS
  audio::poll(stream)
    Test readiness immediately.

  audio::poll(stream, timeoutMs)
    Wait up to timeoutMs for readiness.

PARAMETERS
  stream AS AudioInput or AudioOutput
    An open stream.
  timeoutMs AS Integer
    Maximum wait in milliseconds, in 0..86400000 (timed overload only).

RETURN VALUE
  Boolean
    TRUE when the stream is ready, FALSE otherwise.

ERRORS
  7-705-0018  ErrAudioDevice       The stream failed.

EXAMPLES
  Drive a stream only when it is ready:

  IMPORT audio
  RES mic AS AudioInput = audio::openInput(48000, 1, 512)
  IF audio::poll(mic, 50) THEN
    LET pcm = audio::read(mic, 480, 0)
  END IF
  audio::close(mic)
