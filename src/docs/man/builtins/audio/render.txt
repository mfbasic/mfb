NAME
  render - synthesize a note to raw PCM

SYNOPSIS
  audio::render(note AS AudioNote) AS List OF Byte

PACKAGE
  audio

IMPORTS
  IMPORT audio

DESCRIPTION
  Renders an AudioNote to mono s16le PCM at 48 kHz and returns it as a
  List OF Byte — the same interleaved-sample layout audio::write consumes, so the
  result can be handed straight to a playback stream. One frame is two bytes; the
  returned list is note.noteFrames * 2 bytes long.

  render is a pure MFBASIC synthesizer, not a device call: it never opens hardware
  and never raises. It runs a sine oscillator at note.frequencyHz and shapes it
  with the note's AudioEnvelope — a linear attack from silence to peak (32767), a
  linear decay to sustainLevel, a sustain held for the middle of the note, and a
  linear release over the final releaseFrames. Every sample is scaled by
  note.gainOverall and clamped to the s16 range before being encoded
  little-endian.

PARAMETERS
  note AS AudioNote
    The note to synthesize: frequency, length in frames, amplitude envelope, and
    overall gain. Construct it with AudioNote[...] (and its AudioEnvelope[...]).

RETURN VALUE
  List OF Byte
    Interleaved mono s16le PCM at 48 kHz, note.noteFrames * 2 bytes.

EXAMPLES
  Render one second of A4 (440 Hz) and play it:

  IMPORT audio
  LET env = AudioEnvelope[2400, 4800, 31200, 9600, 12000]
  LET note = AudioNote[440.0, 48000, env, 0.8]
  LET tone = audio::render(note)
  RES out AS AudioOutput = audio::openOutput(48000, 1, 512)
  audio::write(out, tone)
  audio::close(out)
