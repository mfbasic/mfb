NAME
  devices - enumerate the audio devices

SYNOPSIS
  audio::devices() AS List OF AudioDevice

PACKAGE
  audio

IMPORTS
  IMPORT audio

DESCRIPTION
  Returns the audio devices the operating system reports, each as an
  AudioDevice record. The list is never empty on a working host. Each record
  carries an opaque id, a human-readable name, canInput/canOutput capability
  flags, and isDefaultInput/isDefaultOutput flags marking the system defaults.

  The id is a Core Audio device UID on macOS and an ALSA PCM hint NAME on Linux.
  It is opaque: pass it to audio::openInput or audio::openOutput to open that
  specific device; never construct it. A device whose id no longer exists when
  opened (unplugged between devices() and open) raises ErrAudioDevice.

  The records report no channel counts or supported sample rates. Discover a
  working rate/channel combination by attempting to open the device and handling
  the error.

PARAMETERS
  (none)

RETURN VALUE
  List OF AudioDevice
    Every audio device the host reports. Nonempty on a working host; exactly one
    record has isDefaultOutput set when a default output exists, and likewise for
    isDefaultInput.

ERRORS
  7-705-0018  ErrAudioDevice       The device-enumeration API failed.
  7-705-0017  ErrAudioUnavailable  No audio device is present, or (Linux) the
                                   ALSA library could not be resolved.

EXAMPLES
  List every device and mark the defaults:

  IMPORT audio
  IMPORT io
  FOR EACH d IN audio::devices()
    io::print(d.name & " in=" & toString(d.canInput) & " out=" & toString(d.canOutput))
  NEXT
