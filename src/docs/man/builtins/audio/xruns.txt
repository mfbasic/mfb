NAME
  xruns - cumulative overrun/underrun event count

SYNOPSIS
  audio::xruns(stream AS AudioInput) AS Integer
  audio::xruns(stream AS AudioOutput) AS Integer

PACKAGE
  audio

IMPORTS
  IMPORT audio

DESCRIPTION
  Returns the number of xrun events on a stream since it was opened: capture
  overruns for an AudioInput, playback underruns for an AudioOutput. The counter
  is monotonic and increments by exactly one per event.

  It counts events, not lost frames. xruns() > 0 means audio was lost, but the
  amount is unknowable — ALSA does not report how many frames an overrun or
  underrun destroyed, so an event count is the only value that is exact on every
  platform. A stream that never drops audio reports 0.

PARAMETERS
  stream AS AudioInput or AudioOutput
    An open stream.

RETURN VALUE
  Integer
    The cumulative xrun event count since open; 0 when no audio has been lost.

ERRORS
  (none)

EXAMPLES
  Check for lost audio after a playback loop:

  IMPORT audio
  IMPORT io
  RES out AS AudioOutput = audio::openOutput(48000, 2, 512)
  audio::write(out, pcm)
  io::print("underruns: " & toString(audio::xruns(out)))
  audio::close(out)
