NAME
  bytes - UTF-8 encode a String into a response body

SYNOPSIS
  http::bytes(text AS String) AS List OF Byte

PACKAGE
  http

IMPORTS
  IMPORT http

DESCRIPTION
  bytes UTF-8 encodes text into the List OF Byte type used by http::Response.body
  and http::Request.body. It is the String-in convenience for building a body in a
  WITH edit; the constructors http::ok / http::status / http::json encode their
  String body for you. To decode a body back to text, use toString(body).

EXAMPLES
  Setting a response body:

    resp = WITH resp { body := http::bytes("I'm a teapot") }

SEE ALSO
  http::ok, http::responseDefault, strings::toBytes
