NAME
  handleRequest - accept one connection and serve it against a route list

SYNOPSIS
  http::handleRequest(listener AS net::Listener, routes AS List OF Route) AS Nothing
  http::handleRequest(listener AS tls::TlsListener, routes AS List OF Route) AS Nothing

PACKAGE
  http

IMPORTS
  IMPORT http

DESCRIPTION
  handleRequest accepts exactly one connection from listener, reads one full
  HTTP/1.1 request, matches its path (query string stripped first) against routes
  in list order — first match wins — invokes the matched handler, writes the
  returned http::Response, and closes the connection. It is called repeatedly from
  a user-owned DO/LOOP; the listener stays open (it is borrowed, not consumed).

  handleRequest is overloaded by listener type: a net::Listener uses plaintext
  transport, a tls::TlsListener performs the server-side TLS handshake. Both feed
  one shared parse/match/dispatch/emit core, so the route list and handlers are
  identical between the two.

  handleRequest is crash-proof. The handler invocation is wrapped so that a handler
  that fails (any propagated error) is answered with a built-in 500 (text/plain,
  "Internal Server Error") and the loop continues. A path that matches no route is
  answered with a built-in 404 ("Not Found"). A malformed request line or header
  block is answered with 400; a request exceeding the internal 64 MiB cap is
  answered with 413. Connection-level I/O errors (a peer reset mid-read or -write)
  drop the connection and return normally — one bad client never tears down the
  server.

  Content-Length and the status-line reason phrase are always supplied on emit; a
  handler-set Content-Length is ignored to preserve framing, and an empty reason
  is derived from the status code. Connection: close is always sent.

ERRORS
  handleRequest does not itself fail on a bad request: malformed, oversized, and
  handler-failing requests are turned into 400/413/500 responses. Fatal transport
  errors on accept are propagated.

EXAMPLES
  The accept loop:

    RES s AS net::Listener = http::server(8080)
    DO
      http::handleRequest(s, routes)
    LOOP UNTIL FALSE
