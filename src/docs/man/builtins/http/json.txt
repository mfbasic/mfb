NAME
  json - build a 200 application/json response

SYNOPSIS
  http::json(body AS String) AS Response

PACKAGE
  http

IMPORTS
  IMPORT http

DESCRIPTION
  json builds a 200 http::Response whose body is the UTF-8 encoding of body and
  whose Content-Type is "application/json". body is sent verbatim; json does no
  validation or serialization — build the JSON text with the json package (or by
  hand) and pass it here.

EXAMPLES
  A JSON handler:

    FUNC info(req AS http::Request) AS http::Response
      RETURN http::json("{\"ok\":true}")
    END FUNC

SEE ALSO
  http::ok, http::status, http::withHeader
