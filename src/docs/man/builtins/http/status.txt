NAME
  status - build a text/plain response with an arbitrary status code

SYNOPSIS
  http::status(code AS Integer, body AS String) AS Response

PACKAGE
  http

IMPORTS
  IMPORT http

DESCRIPTION
  status builds an http::Response with the given status code, a text/plain body
  (the UTF-8 encoding of body), and Content-Type "text/plain; charset=utf-8". The
  ok field of the returned response is TRUE when code is in 200..299. The reason
  phrase is left empty and is derived from the status code by handleRequest on
  emit.

EXAMPLES
  A custom error response:

    RETURN http::status(422, "validation failed")

SEE ALSO
  http::ok, http::json, http::responseDefault, http::withHeader
