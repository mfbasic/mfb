NAME
  printError - write text to standard error followed by a newline

SYNOPSIS
  io::printError(value AS String) AS Nothing

PACKAGE
  io

IMPORTS
  IMPORT io

DESCRIPTION
  Writes value to the process standard error stream and then appends a single
  newline character. The value is treated as UTF-8 text and is emitted exactly
  as stored, byte for byte, with no further encoding, escaping, or newline
  translation beyond the one trailing newline this function adds. Any valid
  Unicode scalar values are permitted, including multi-byte characters and emoji.

  Only String values are accepted. The function performs no implicit
  conversion, so to report a non-string value you must convert it first, for
  example with toString. An empty String prints nothing but the trailing
  newline.

  Output is directed to whichever destination is bound to standard error. In a
  normal console program this is file descriptor 2; in app mode the same call is
  routed to the application transcript instead of a file descriptor. The write
  is the function's only side effect. Standard error is written immediately and
  is never buffered, so error output can never sit unseen in a buffer. To write
  without a trailing newline, use io::writeError; to send text to standard output
  instead, use io::print.

PARAMETERS
  value AS String
    The text to write. Interpreted as UTF-8 and emitted unchanged. May be any
    String, including the empty String and strings containing arbitrary Unicode
    characters.

RETURN VALUE
  AS Nothing
    Returns nothing. The function is invoked for its side effect of writing to
    standard error.

ERRORS
  77020002 (ErrOutput)
  Raised when the underlying write to standard error fails, for example when the
  error stream is closed or a host write error occurs.

EXAMPLES
  Report an error message:

    io::printError("failed")

  Print Unicode text including emoji:

    io::printError("é日😀")

  Convert a non-string value before printing:

    io::printError(toString(42))

SEE ALSO
  io::writeError, io::print, io::flush
