NAME
  pollInput - test whether standard input is ready to read without blocking

SYNOPSIS
  io::pollInput() AS Boolean
  io::pollInput(timeoutMs AS Integer = 0) AS Boolean

PACKAGE
  io

IMPORTS
  IMPORT io

DESCRIPTION
  Waits until standard input (file descriptor 0) can be read without blocking,
  returning TRUE when input becomes ready and FALSE when the wait elapses first.
  Internally a single poll on the read end of standard input is performed, so the
  call reports readiness only; it does not consume any input. A following read
  with io::readLine, io::readChar, io::readByte, or io::input retrieves the bytes.

  The timeoutMs argument controls how long the call waits:
    timeoutMs < 0 waits indefinitely until input is ready.
    timeoutMs = 0 performs a nonblocking readiness check and returns immediately.
    timeoutMs > 0 waits up to that many milliseconds for input to become ready.

  When the argument is omitted it defaults to 0, so io::pollInput() performs a
  nonblocking readiness check.

  Readiness includes end of file: when standard input has reached EOF, the poll
  reports the descriptor as readable and the call returns TRUE, after which the
  next read reports end of file. Standard input remains under the host terminal's
  normal line discipline; on a canonical (cooked-mode) terminal, input may not be
  reported as ready until the user submits a complete line with Return.

OVERLOADS
  io::pollInput() AS Boolean
    Performs a nonblocking readiness check, equivalent to io::pollInput(0). The
    call returns immediately with TRUE if input is already available and FALSE
    otherwise.

  io::pollInput(timeoutMs AS Integer = 0) AS Boolean
    Waits up to timeoutMs milliseconds for input to become ready, waiting
    indefinitely when timeoutMs is negative and not at all when it is zero.

PARAMETERS
  timeoutMs AS Integer
    The maximum time to wait, in milliseconds. A negative value waits
    indefinitely until input is ready; zero performs an immediate nonblocking
    check; a positive value bounds the wait to that many milliseconds. When
    omitted, defaults to 0.

RETURN VALUE
  AS Boolean
    TRUE when standard input is ready to read (including when it has reached end
    of file) before the timeout elapses, and FALSE when the wait times out with
    no input available. A zero timeout returns FALSE immediately when nothing is
    pending.

ERRORS
  77020005 (ErrInput)
  Raised when polling standard input fails.

EXAMPLES
  Read a line only when input is already pending:

    IF io::pollInput() THEN
      io::print(io::readLine())
    END IF

  Wait up to one second for a keypress, otherwise report a timeout:

    IF io::pollInput(1000) THEN
      io::print(io::readChar())
    ELSE
      io::print("timeout")
    END IF

  Block until input is available, then read a byte:

    IF io::pollInput(-1) THEN
      io::print(toString(io::readByte()))
    END IF

SEE ALSO
