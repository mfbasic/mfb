NAME
  isBuffered - test whether standard-output buffering is enabled

SYNOPSIS
  io::isBuffered() AS Boolean

PACKAGE
  io

IMPORTS
  IMPORT io

DESCRIPTION
  Reports whether opt-in standard-output buffering is currently enabled for the
  calling thread. Returns TRUE after io::setBuffered(TRUE) and FALSE otherwise.
  Buffering is off by default, so a program that never calls io::setBuffered
  always observes FALSE here.

  Buffering is a per-thread setting: each thread has its own standard-output
  buffer and its own enabled flag, so a call on one thread does not observe the
  state of another. Standard error is never buffered and has no corresponding
  control.

  The function takes no arguments and never fails; it only reads the current
  buffering state and writes no output.

  Platform note: in app mode standard output is rendered by the application
  window and is never buffered, so this call always returns FALSE.

PARAMETERS
  None.

RETURN VALUE
  AS Boolean
    Returns TRUE when standard-output buffering is enabled for this thread and
    FALSE otherwise (including when it has never been enabled). In app mode the
    result is always FALSE.

ERRORS
  No errors.

EXAMPLES
  Enable buffering only when it is not already on:

    IF NOT io::isBuffered() THEN
      io::setBuffered(TRUE)
    END IF

  Capture the current state to restore it later:

    LET wasBuffered AS Boolean = io::isBuffered()

SEE ALSO
