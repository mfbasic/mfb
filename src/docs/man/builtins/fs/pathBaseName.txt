NAME
  pathBaseName - return the final component of a path

SYNOPSIS
  fs::pathBaseName(path AS String) AS String

PACKAGE
  fs

IMPORTS
  IMPORT fs

DESCRIPTION
  Returns the final component of path, that is, the part after the last "/"
  separator. The operation is purely syntactic: it inspects the bytes of path
  and never consults the filesystem, resolves "." or ".." segments, follows
  symbolic links, or checks whether any path exists.

  Trailing "/" separators are ignored before the final component is located, so
  "target/output/" and "target/output" both yield "output". After trimming, the
  bytes are scanned backward for the last "/" separator and everything following
  it becomes the result, so the returned String carries no leading separator.

  When path contains no separator, it is returned unchanged. When path is "/"
  itself, or consists only of "/" separators, "/" is returned. An empty path
  returns an empty String.

  The scan is byte-oriented, so UTF-8 file names are preserved unchanged and any
  embedded bytes are treated literally. A new String holding the final-component
  bytes is allocated for the result. The function has no side effects other than
  allocating the returned String.

PARAMETERS
  path AS String
    The path whose final component is wanted. Interpreted as raw UTF-8 bytes.
    Trailing "/" separators are ignored. May be empty.

RETURN VALUE
  AS String
    The final component of path, with no leading separator. Returns path
    unchanged when it has no separator, "/" when path is "/" or consists only of
    separators, and an empty String when path is empty.

ERRORS
  77010001 (ErrOutOfMemory)
  Raised when the result String for the final component cannot be allocated.

EXAMPLES
  A directory and a file name yield the file name:

    io::print(fs::pathBaseName("target/output.txt"))

  The final component of an absolute path:

    io::print(fs::pathBaseName("/usr/local/bin"))

  A trailing separator is ignored before the component is located:

    io::print(fs::pathBaseName("/usr/local/bin/"))

  A path with no separator is returned unchanged:

    io::print(fs::pathBaseName("output.txt"))

  The root path yields itself:

    io::print(fs::pathBaseName("/"))
