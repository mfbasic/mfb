# ln10

The mathematical constant `ln(10)` as a `Float`, the natural logarithm of 10.

## Synopsis

```
math::ln10 AS Float
```

## Package

math

## Imports

```
IMPORT math
```

`math` is a built-in package, so no manifest dependency is required.
[[src/builtins/math.rs:is_math_constant]]

## Description

`math::ln10` is a constant, not a callable function. It takes no arguments and
no parentheses: write the name wherever a `Float` expression is expected.
[[src/builtins/math.rs:constant_type_name]]

`math::ln10` is the closest 64-bit IEEE 754 double-precision `Float` to `ln(10)`, which is irrational and has no exact finite binary representation. Its decimal shorthand is `2.302585092994046`. [[src/builtins/math.rs:constant_value]]

The constant is a compile-time value: it is substituted at the point of use, performs no computation and has no side effects, and evaluates to the same bit pattern on every reference and on every target. [[src/builtins/math.rs:is_math_constant]]

The same constant is also available as a `Fixed` under the name
`math::ln10Fixed`. There is no automatic conversion between the two forms, so pick
the one whose type matches the expression you are writing.
[[src/builtins/math.rs:constant_type_name]]

## Parameters

`math::ln10` is a constant and takes no parameters.
[[src/builtins/math.rs:is_math_constant]]

## Return value

| Type | Description |
| --- | --- |
| `Float` | The `Float` nearest to `ln(10)`, approximately `2.302585092994046`. The same value on every reference. [[src/builtins/math.rs:constant_value]] |

## Errors

No errors. Referencing a constant performs no computation, so there is no failure path. [[src/builtins/math.rs:constant_value]]

## Examples

Read the constant into a `Float` binding:

```
IMPORT math
IMPORT io

SUB main()
  LET value AS Float = math::ln10
  io::print(toString(value))
END SUB
```

Convert a natural logarithm to base 10:

```
IMPORT math
IMPORT io

SUB main()
  LET x AS Float = 1000.0
  LET log10 AS Float = math::log(x) / math::ln10
  io::print(toString(log10))
END SUB
```
