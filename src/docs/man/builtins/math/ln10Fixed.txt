NAME
  ln10Fixed - mathematical constant ln(10) as Fixed

SYNOPSIS
  math::ln10Fixed AS Fixed

PACKAGE
  math

IMPORTS
  IMPORT math

DESCRIPTION
  The natural logarithm of 10, expressed as a Fixed value. Its value is the
  closest Fixed to ln(10), approximately 2.302585092994046. The exact
  mathematical value is irrational and cannot be represented exactly, so this
  constant is the nearest representable Fixed.

  ln10Fixed is a constant, not a callable function: it takes no arguments and is
  referenced by name wherever a Fixed expression is expected. It performs no
  computation at the call site and has no side effects, evaluating to the same
  Fixed value on every use and on every platform. This makes it suitable for
  deterministic decimal arithmetic where a Float is not desired.

  This constant is commonly used to convert between natural and base-10
  logarithms: the base-10 logarithm of a value x equals
  math::log(x) / math::ln10Fixed. Use math::ln10 for the Float form of the same
  constant, and math::ln2Fixed for the natural logarithm of 2.

PARAMETERS
  None.
    ln10Fixed is a constant value and accepts no arguments.

RETURN VALUE
  AS Fixed
    The closest Fixed value to ln(10), approximately 2.302585092994046. The same
    value is produced on every reference.

ERRORS
  No errors.

EXAMPLES
  Read the constant into a Fixed:

    IMPORT math

    LET value AS Fixed = math::ln10Fixed

  Compute a base-10 logarithm from the natural logarithm:

    IMPORT math

    LET log10 AS Fixed = math::log(x) / math::ln10Fixed
