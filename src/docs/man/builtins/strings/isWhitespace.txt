NAME
  isWhitespace - test whether a scalar is whitespace

SYNOPSIS
  strings::isWhitespace(scalar AS Scalar) AS Boolean

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  isWhitespace reports whether scalar is a Unicode whitespace scalar (a Z* separator, or one of tab, line feed, vertical tab, form feed, carriage return, or U+0085). It is total: it returns a Boolean for
  every Scalar and never fails. Classification follows the Unicode
  general-category database (category White_Space).

  To test a whole string, walk it with strings::toScalars and fold the results;
  the question "is this string all letters/digits" is deliberately left to the
  caller, since isWhitespace classifies a single scalar.

PARAMETERS
  scalar AS Scalar
    The Unicode scalar to classify. Any Scalar is accepted.

RETURN VALUE
  AS Boolean
    TRUE when scalar is whitespace, FALSE otherwise.

ERRORS
  No errors.

EXAMPLES
  Classify individual scalars:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(toString(strings::isWhitespace(`\t`)))
      io::print(toString(strings::isWhitespace(`x`)))
      RETURN 0
    END FUNC
