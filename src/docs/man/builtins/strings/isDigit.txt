NAME
  isDigit - test whether a scalar is a decimal digit

SYNOPSIS
  strings::isDigit(scalar AS Scalar) AS Boolean

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  isDigit reports whether scalar is a Unicode decimal digit (general category Nd). It is total: it returns a Boolean for
  every Scalar and never fails. Classification follows the Unicode
  general-category database (category Nd).

  To test a whole string, walk it with strings::toScalars and fold the results;
  the question "is this string all letters/digits" is deliberately left to the
  caller, since isDigit classifies a single scalar.

PARAMETERS
  scalar AS Scalar
    The Unicode scalar to classify. Any Scalar is accepted.

RETURN VALUE
  AS Boolean
    TRUE when scalar is a decimal digit, FALSE otherwise.

ERRORS
  No errors.

EXAMPLES
  Classify individual scalars:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(toString(strings::isDigit(`7`)))
      io::print(toString(strings::isDigit(`x`)))
      RETURN 0
    END FUNC
