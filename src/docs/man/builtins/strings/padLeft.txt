NAME
  padLeft - left-pad a string to a given width

SYNOPSIS
  strings::padLeft(value AS String, width AS Integer, padChar AS String = " ") AS String

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  Returns value padded on the left until it spans a total of width Unicode
  scalar values. Padding is performed by prepending copies of padChar one at a
  time; the number of copies prepended is width minus the current scalar length
  of value. Length is measured in Unicode scalar values, not UTF-8 bytes or
  user-perceived grapheme clusters, so a multi-byte scalar counts as one toward
  the width.

  When the scalar length of value is already greater than or equal to width, no
  padding is added and value is returned unchanged; the function never truncates
  value to fit within width.

  The padChar argument is optional and defaults to a single space (" "). When
  supplied, it must be exactly one Unicode scalar value. The function does not
  mutate its arguments; it returns a newly built String.

PARAMETERS
  value AS String
    The string to pad. Returned unchanged when its scalar length is already at
    least width.

  width AS Integer
    The target total length in Unicode scalar values. Must be zero or positive;
    a value of 0 means no padding is ever added. A negative width raises an
    error.

  padChar AS String
    The fill character prepended to reach width. Optional; defaults to a single
    space. Must consist of exactly one Unicode scalar value (neither empty nor
    more than one scalar).

RETURN VALUE
  AS String
    A new String equal to value prefixed with enough copies of padChar to reach
    width scalar values. When value is already at least width scalars long, the
    original value is returned unchanged.

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when width is negative, or when padChar is not exactly one Unicode
  scalar value (empty or longer than one scalar).

EXAMPLES
  Left-pad a number with zeros to a width of 5:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(strings::padLeft("42", 5, "0"))
      RETURN 0
    END FUNC

  Using the default space pad character:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(strings::padLeft("hi", 6))
      RETURN 0
    END FUNC
