NAME
  repeat - concatenate a string with itself a given number of times

SYNOPSIS
  strings::repeat(value AS String, times AS Integer) AS String

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  repeat returns a new String formed by concatenating value with itself times
  times. The result is value written out end to end times consecutive copies,
  with nothing inserted between the copies.

  Copying is performed on the raw UTF-8 bytes of value, so any multi-byte
  scalar or grapheme cluster is reproduced intact in every copy; repeat never
  splits a character. The length of the result in bytes is the byte length of
  value multiplied by times.

  A times of 0 returns the empty string, regardless of value. A times of 1
  returns a copy equal to value. times must not be negative.

  repeat does not mutate value and has no side effects. The returned String is
  a new owned value.

PARAMETERS
  value AS String
    The string to repeat. Any String, including the empty String, is accepted;
    repeating the empty String yields the empty String for any valid times.

  times AS Integer
    The number of copies to concatenate. Must be 0 or greater. A value of 0
    yields the empty string and a value of 1 yields a copy of value.

RETURN VALUE
  AS String
    A new String containing times consecutive copies of value. Returns the
    empty string when times is 0 or when value is the empty String.

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when times is negative.

TYPE CHECKING
  value must be String. times must be an Integer.

EXAMPLES
  Repeat a short string three times:

    LET line AS String = strings::repeat("ab", 3)

  A count of zero yields the empty string:

    LET empty AS String = strings::repeat("ab", 0)

  Build a separator rule:

    LET rule AS String = strings::repeat("-", 40)
