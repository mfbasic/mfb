NAME
  fromScalars - build a string from Unicode scalars

SYNOPSIS
  strings::fromScalars(scalars AS List OF Scalar) AS String

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  fromScalars encodes a List of Scalar into a String, concatenating the
  one-scalar UTF-8 encoding of each element in order. It is the inverse of
  strings::toScalars: the round trip
  strings::fromScalars(strings::toScalars(s)) equals s for every String s.

  Because every Scalar is a valid, non-surrogate Unicode code point, encoding
  always succeeds. An empty List yields the empty string. The input List is not
  modified; the returned String is a fresh value.

PARAMETERS
  scalars AS List OF Scalar
    The scalars to encode, in order. Any List of Scalar is accepted, including
    the empty List.

RETURN VALUE
  AS String
    The UTF-8 string formed by concatenating each scalar's encoding. The empty
    List yields "".

ERRORS
  No errors. Every Scalar is a valid code point, so encoding always succeeds.

EXAMPLES
  Rebuild a string from scalars:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      LET s = strings::fromScalars(strings::toScalars("héllo"))
      io::print(s)
      RETURN 0
    END FUNC

  Build a string from literal scalars:

    IMPORT io
    IMPORT strings
    IMPORT collections

    FUNC main AS Integer
      MUT xs AS List OF Scalar = [`h`, `i`, `!`]
      io::print(strings::fromScalars(xs))
      RETURN 0
    END FUNC
