NAME
  isUpper - test whether a scalar is an uppercase letter

SYNOPSIS
  strings::isUpper(scalar AS Scalar) AS Boolean

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  isUpper reports whether scalar is an uppercase letter (general category Lu). It is total: it returns a Boolean for
  every Scalar and never fails. Classification follows the Unicode
  general-category database (category Lu).

  To test a whole string, walk it with strings::toScalars and fold the results;
  the question "is this string all letters/digits" is deliberately left to the
  caller, since isUpper classifies a single scalar.

PARAMETERS
  scalar AS Scalar
    The Unicode scalar to classify. Any Scalar is accepted.

RETURN VALUE
  AS Boolean
    TRUE when scalar is an uppercase letter, FALSE otherwise.

ERRORS
  No errors.

EXAMPLES
  Classify individual scalars:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(toString(strings::isUpper(`Q`)))
      io::print(toString(strings::isUpper(`q`)))
      RETURN 0
    END FUNC
