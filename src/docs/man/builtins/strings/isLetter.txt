NAME
  isLetter - test whether a scalar is a Unicode letter

SYNOPSIS
  strings::isLetter(scalar AS Scalar) AS Boolean

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  isLetter reports whether scalar is a Unicode letter (general categories Lu, Ll, Lt, Lm, Lo). It is total: it returns a Boolean for
  every Scalar and never fails. Classification follows the Unicode
  general-category database (category L*).

  To test a whole string, walk it with strings::toScalars and fold the results;
  the question "is this string all letters/digits" is deliberately left to the
  caller, since isLetter classifies a single scalar.

PARAMETERS
  scalar AS Scalar
    The Unicode scalar to classify. Any Scalar is accepted.

RETURN VALUE
  AS Boolean
    TRUE when scalar is a Unicode letter, FALSE otherwise.

ERRORS
  No errors.

EXAMPLES
  Classify individual scalars:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(toString(strings::isLetter(`A`)))
      io::print(toString(strings::isLetter(`5`)))
      RETURN 0
    END FUNC
