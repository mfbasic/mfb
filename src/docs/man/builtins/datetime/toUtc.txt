NAME
  toUtc - project an Instant into UTC to produce a civil DateTime

SYNOPSIS
  datetime::toUtc(at AS Instant) AS DateTime

PACKAGE
  datetime

IMPORTS
  IMPORT datetime

DESCRIPTION
  datetime::toUtc projects the absolute instant at into the UTC zone, yielding
  the calendar date and wall-clock time that an observer reading UTC sees at that
  moment. It is exactly shorthand for datetime::inZone(at, datetime::utc()): the
  resolved UTC offset is always zero, so the instant's seconds-since-epoch are
  split directly into a civil year/month/day (proleptic Gregorian calendar) and an
  hour/minute/second-of-day with no offset adjustment.

  The returned DateTime carries the civil date, the civil time, the UTC zone, and
  a resolved offset of zero. Because the zero offset is pinned on the result, the
  DateTime round-trips back to the original instant via datetime::resolve with no
  further zone lookup. The instant's sub-second nanos field is preserved verbatim
  into the time's nanos field; only the seconds field participates in the date and
  time computation. May name an instant before the Unix epoch.

  Unlike datetime::toLocal, datetime::toUtc is pure: it reads no host time-zone
  configuration and produces the same result on every platform. Because the offset
  is always zero, the projection cannot overflow the Integer range.

PARAMETERS
  at AS Instant
    The absolute point on the UTC timeline to project. Its seconds field (seconds
    since the Unix epoch) drives the civil date/time computation; its nanos field
    is copied unchanged into the result's time. May name an instant before the
    epoch.

RETURN VALUE
  AS DateTime
    A DateTime holding the civil date and wall-clock time observed in UTC at the
    instant at, together with the UTC zone and a resolved offset of zero seconds.
    The nanos of the time equal the nanos of at. The result resolves back to at via
    datetime::resolve.

ERRORS
  No errors.

EXAMPLES
  Project the current instant into UTC:

    IMPORT datetime

    LET dt AS DateTime = datetime::toUtc(datetime::now())

  Round-trip an instant through UTC and back:

    IMPORT datetime

    LET at AS Instant = datetime::now()
    LET dt AS DateTime = datetime::toUtc(at)
    LET back AS Instant = datetime::resolve(dt)

SEE ALSO
  datetime::inZone, datetime::toLocal, datetime::utc, datetime::resolve
