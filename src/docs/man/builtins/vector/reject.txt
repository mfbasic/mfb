NAME
  reject - component of a orthogonal to b

SYNOPSIS
  vector::reject(a AS T_N, b AS T_N) AS T_N

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns a - project(a, b): the part of a perpendicular to b. Same
  zero-b error and per-type rounding as project; for Integer, project +
  reject round-trips as closely as integers allow.

PARAMETERS
  a AS T_N
    the vector to decompose

  b AS T_N
    the direction to remove; must be non-zero

RETURN VALUE
  AS T_N
    the orthogonal component

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when b has zero length.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Orthogonal component relative to the +x axis:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::reject(vector::Float2[2.0,2.0], vector::Float2[1.0,0.0])))
      RETURN 0
    END FUNC
