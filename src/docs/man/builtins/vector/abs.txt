NAME
  abs - component-wise absolute value

SYNOPSIS
  vector::abs(v AS T_N) AS T_N

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns the component-wise absolute value, each component math::abs(v.i).
  Integer or Fixed abs of the minimum representable value traps
  ErrOverflow, exactly as the scalar math::abs does.

PARAMETERS
  v AS T_N
    the vector

RETURN VALUE
  AS T_N
    the component-wise absolute value

ERRORS
  77050010 (ErrOverflow)
  Raised when an Integer or Fixed result exceeds the representable range,
  as in the scalar math:: functions.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Component-wise absolute value:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::abs(vector::Float2[-2.0, 3.0])))
      RETURN 0
    END FUNC

  min, max, scale
