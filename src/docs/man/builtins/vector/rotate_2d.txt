NAME
  rotate_2d - rotate a 2D vector counterclockwise by an angle

SYNOPSIS
  vector::rotate_2d(v AS T2, angle AS Float) AS T2

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  2D only. Rotates v counterclockwise by angle radians:
  (v.x*cos - v.y*sin, v.x*sin + v.y*cos). angle is a Float for every
  element type; Fixed and Integer convert it with toFixed and use the
  deterministic Q32.32 sin/cos, with Integer rounding each component half
  away from zero. Float uses the in-tree Float sin/cos.

PARAMETERS
  v AS T2
    a 2D vector (Float2, Fixed2, or Integer2)

  angle AS Float
    rotation angle in radians, counterclockwise

RETURN VALUE
  AS T2
    the rotated vector

ERRORS
  77050010 (ErrOverflow)
  Raised when an Integer or Fixed result exceeds the representable range,
  as in the scalar math:: functions.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Rotate the +x axis by a right angle:

    IMPORT vector
    IMPORT io
    IMPORT math
    FUNC main AS Integer
      io::print(toString(vector::rotate_2d(vector::Float2[1.0, 0.0], math::pi2)))
      RETURN 0
    END FUNC
