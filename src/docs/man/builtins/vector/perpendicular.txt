NAME
  perpendicular - 2D left perpendicular (-y, x)

SYNOPSIS
  vector::perpendicular(v AS T2) AS T2

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  2D only. Returns the left perpendicular (-v.y, v.x), a 90-degree
  counterclockwise rotation. This is the named form of the unary 2D
  cross and shares its implementation. Exact for every element type.

PARAMETERS
  v AS T2
    a 2D vector (Float2, Fixed2, or Integer2)

RETURN VALUE
  AS T2
    the left perpendicular

ERRORS
  None.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Perpendicular of the +x axis:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::perpendicular(vector::Float2[1.0, 0.0])))
      RETURN 0
    END FUNC

SEE ALSO
  cross, rotate_2d
