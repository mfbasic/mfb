NAME
  cross - generalized (n-1)-ary cross product

SYNOPSIS
  vector::cross(v AS T2) AS T2
  vector::cross(a AS T3, b AS T3) AS T3
  vector::cross(a AS T4, b AS T4, c AS T4) AS T4

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns the cross product: the vector perpendicular to its operands.
  This is the generalized (n-1)-ary form, so its arity is
  dimension-specific. In 2D it is unary and returns the left
  perpendicular (-v.y, v.x) (a 90-degree counterclockwise rotation). In
  3D it is the binary a x b. In 4D it is ternary: the vector
  perpendicular to all three operands, via the cofactor determinant.
  Exact for every element type (no rounding).

OVERLOADS
  vector::cross(v AS T2) AS T2
  vector::cross(a AS T3, b AS T3) AS T3
  vector::cross(a AS T4, b AS T4, c AS T4) AS T4
    Resolved by argument record type and arity; T is the element type
    (Float, Fixed, or Integer) and N the dimension (2, 3, or 4).

PARAMETERS
  v / a AS T_N
    the operand(s); one in 2D, two in 3D, three in 4D

RETURN VALUE
  AS T_N
    the perpendicular vector

ERRORS
  77050010 (ErrOverflow)
  Raised when an Integer or Fixed result exceeds the representable range,
  as in the scalar math:: functions.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  3D cross product of two basis vectors:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::cross(vector::Float3[1.0,0.0,0.0], vector::Float3[0.0,1.0,0.0])))
      RETURN 0
    END FUNC
