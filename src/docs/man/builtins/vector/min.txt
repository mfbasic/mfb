NAME
  min - component-wise minimum

SYNOPSIS
  vector::min(a AS T_N, b AS T_N) AS T_N

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns the component-wise minimum, each component math::min(a.i, b.i).

PARAMETERS
  a AS T_N
    the first vector

  b AS T_N
    the second vector, same type as a

RETURN VALUE
  AS T_N
    the component-wise minimum

ERRORS
  None.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Component-wise minimum:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::min(vector::Float2[2.0,3.0], vector::Float2[4.0,1.0])))
      RETURN 0
    END FUNC
