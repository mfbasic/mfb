NAME
  toAddress - project a Url onto a connectable Address

SYNOPSIS
  net::toAddress(url AS Url) AS Address

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  toAddress projects a parsed Url onto a connectable endpoint, an Address whose host
  is url.host and whose port is url.port. Because net::toUrl always fills port with
  the scheme default when absent, the result is concrete and feeds the
  net::connectTcp(address AS Address) overload directly. toAddress performs no DNS
  resolution; it carries the host name through unchanged. Use net::lookup when a
  resolved IP address is required.

  Not yet provided in this version: connect directly with
  net::connectTcp(url.host, url.port), which is equivalent for the plaintext and
  TLS clients. A follow-up adds the native shim that constructs the Address record.

ERRORS
  None for the projection itself; connection errors are raised by net::connectTcp.

EXAMPLES
    IMPORT net

    LET u = net::toUrl("http://example.com/")
    ' Equivalent to net::connectTcp(net::toAddress(u)):
    RES c = net::connectTcp(u.host, u.port)

SEE ALSO
  net::toUrl, net::connectTcp, net::lookup
