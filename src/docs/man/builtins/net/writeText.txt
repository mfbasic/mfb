NAME
  writeText - write a String to a connected socket as UTF-8 text

SYNOPSIS
  net::writeText(sock AS Socket, value AS String) AS Nothing

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  Sends the UTF-8 bytes of value over the connected stream Socket sock and
  returns nothing. The String's packed byte data is written directly: a String
  already holds well-formed UTF-8, so the bytes are written exactly as held, with
  no re-encoding, decoding, or newline translation.

  writeText writes the entire String before returning. It loops internally,
  advancing past the bytes that each underlying write accepted and reissuing the
  write for the remainder, so a short host write that transfers only part of the
  buffer is resumed rather than treated as complete. When the call returns
  successfully, every byte of value has been handed to the socket's send buffer;
  this does not guarantee the peer has received or read them. An empty String
  writes nothing and returns immediately.

  The socket must still be open. writeText blocks while the send buffer is full
  until space becomes available or the socket's write timeout elapses; if the
  timeout elapses before all bytes are sent, the call raises ErrWriteTimeout. Use
  net::setWriteTimeout to bound how long a write may wait. Use net::write instead
  to send raw binary data from a List OF Byte rather than UTF-8 text from a
  String.

PARAMETERS
  sock AS Socket
    The connected stream socket to write to, as returned by net::connectTcp or
    net::accept. Must not have been closed.

  value AS String
    The text to send. Written verbatim as the String's UTF-8 bytes, in order. An
    empty String sends nothing.

RETURN VALUE
  AS Nothing
    Returns nothing on success, after every byte of value has been written to the
    socket's send buffer.

ERRORS
  77030004 (ErrResourceClosed)
  Raised when sock has already been closed before the call.

  77070004 (ErrConnectionClosed)
  Raised when the underlying write fails because the peer has closed the
  connection or the socket is otherwise no longer usable.

  77070006 (ErrWriteTimeout)
  Raised when the socket's write timeout elapses before all bytes of value could
  be sent.

EXAMPLES
  Send an HTTP request as text over a connected socket:

    IMPORT net

    RES client = net::connectTcp("example.com", 80)
    net::writeText(client, "GET / HTTP/1.0\r\n\r\n")
    LET reply = net::readText(client, 1024)
    ' client is closed by lexical drop when this scope ends

  Echo text received from a peer back to it:

    IMPORT net

    RES server = net::listenTcp("127.0.0.1", 8080)
    RES peer = net::accept(server)
    DO
      LET chunk = net::readText(peer, 4096)
      IF len(chunk) = 0 THEN
        EXIT DO
      END IF
      net::writeText(peer, chunk)
    LOOP
    ' peer and server are closed by lexical drop when this scope ends
