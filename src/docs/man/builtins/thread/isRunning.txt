NAME
  isRunning - test whether a worker is still running

SYNOPSIS
  thread::isRunning(t AS Thread OF Msg TO Out) AS Boolean

PACKAGE
  thread

IMPORTS
  IMPORT thread

DESCRIPTION
  Reports whether the worker behind a parent Thread handle is still executing
  its entry function. The call locks the thread's outbound queue, reads the
  shared worker state, and unlocks before returning, so it observes a coherent
  snapshot of the worker's progress without consuming any messages or the
  worker result.

  The result reflects three internal states. While the entry function is still
  running, the call returns TRUE. Once the worker has finished — whether it
  completed normally and stored its Out result or failed and stored an Err —
  the call returns FALSE; the stored result remains available for a later
  thread::waitFor(t) or t.result. If the parent Thread handle has already been
  closed, the call instead raises ErrResourceClosed. A handle is closed after
  thread::waitFor(t) or t.result retrieves the one-shot result, so polling a
  Thread with isRunning after the result has been taken fails rather than
  returning FALSE.

  isRunning only accepts a parent Thread handle, not a ThreadWorker; worker code
  observes cancellation with thread::isCancelled instead. The call does not
  block, request cancellation, or otherwise change the worker's state.

PARAMETERS
  t AS Thread OF Msg TO Out
    The parent Thread handle returned by thread::start, where Msg is the inbound
    message type and Out is the worker's result type. Must be a live, unretrieved
    handle.

RETURN VALUE
  AS Boolean
    TRUE while the worker entry function is still running; FALSE after the worker
    has completed or failed and stored its result (but before the result handle
    is closed).

ERRORS
  77030004 (ErrResourceClosed)
  Raised when the parent Thread handle has already been closed, such as after
  thread::waitFor(t) or t.result has retrieved the worker result.

EXAMPLES
  Poll a worker until it finishes, then collect its result:

    LET t AS Thread OF Nothing TO Integer = thread::start(work, NOTHING)
    WHILE thread::isRunning(t)
      io::print("still working...")
    ENDWHILE
    LET count AS Integer = thread::waitFor(t)

SEE ALSO
  thread::start, thread::waitFor, thread::poll, thread::isCancelled
