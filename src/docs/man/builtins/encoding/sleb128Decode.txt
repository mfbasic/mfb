NAME
  sleb128Decode - decode signed LEB128

SYNOPSIS
  encoding::sleb128Decode(data AS List OF Byte) AS Integer

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes signed LEB128 bytes to a signed Integer. Fails on overflow or truncated data.

PARAMETERS
  data AS List OF Byte
    The LEB128 bytes.

RETURN VALUE
  AS Integer
    The decoded value.

ERRORS
  77050003 (ErrInvalidFormat)
  On overflow or truncation.

EXAMPLES
  IMPORT encoding

  encoding::sleb128Decode(bytes)

SEE ALSO
  mfb man encoding, mfb man bits
