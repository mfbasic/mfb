NAME
  punycodeDecode - decode Punycode to a hostname

SYNOPSIS
  encoding::punycodeDecode(asciiDomain AS String) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Translates an ASCII Punycode hostname back to its Unicode form. Fails on malformed Punycode.

PARAMETERS
  asciiDomain AS String
    The ASCII Punycode hostname.

RETURN VALUE
  AS String
    The Unicode hostname.

ERRORS
  77050003 (ErrInvalidFormat)
  On malformed Punycode.

EXAMPLES
  IMPORT encoding

  encoding::punycodeDecode("xn--mnchen-3ya.de")
