NAME
  base64UrlEncode - encode bytes to URL-safe Base64

SYNOPSIS
  encoding::base64UrlEncode(data AS List OF Byte) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Encodes data to URL-safe Base64 (RFC 4648 5) using -_ and no padding. Total.

PARAMETERS
  data AS List OF Byte
    The bytes to encode.

RETURN VALUE
  AS String
    The URL-safe Base64 string.

ERRORS
  None.

EXAMPLES
  IMPORT encoding

  encoding::base64UrlEncode(raw)

SEE ALSO
  mfb man encoding, mfb man bits
