NAME
  uleb128Encode - encode an unsigned LEB128 integer

SYNOPSIS
  encoding::uleb128Encode(value AS Integer) AS List OF Byte

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Encodes a non-negative integer using unsigned LEB128. Fails when value is negative.

PARAMETERS
  value AS Integer
    The non-negative value to encode.

RETURN VALUE
  AS List OF Byte
    The LEB128 bytes.

ERRORS
  77050003 (ErrInvalidFormat)
  When value is negative.

EXAMPLES
  IMPORT encoding

  encoding::uleb128Encode(624485)

  mfb man encoding, mfb man bits
