NAME
  formUrlDecode - decode form-urlencoded text

SYNOPSIS
  encoding::formUrlDecode(text AS String) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes application/x-www-form-urlencoded text, turning + back into spaces and decoding %XX escapes as UTF-8. Fails on malformed input.

PARAMETERS
  text AS String
    The form-encoded text.

RETURN VALUE
  AS String
    The decoded text.

ERRORS
  77050003 (ErrInvalidFormat)
  On a malformed escape or bad UTF-8.

EXAMPLES
  IMPORT encoding

  encoding::formUrlDecode("a+b")

SEE ALSO
  mfb man encoding, mfb man bits
