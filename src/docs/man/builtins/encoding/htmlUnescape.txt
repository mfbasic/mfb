NAME
  htmlUnescape - decode HTML entities

SYNOPSIS
  encoding::htmlUnescape(text AS String) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes named (such as &eacute;) and numeric (&#233; or &#xE9;) HTML entities back to their characters. Fails on a malformed entity structure or unknown name.

PARAMETERS
  text AS String
    The text to decode.

RETURN VALUE
  AS String
    The decoded text.

ERRORS
  77050003 (ErrInvalidFormat)
  On a malformed or unknown entity.

EXAMPLES
  IMPORT encoding

  encoding::htmlUnescape("&lt;a&gt;")
