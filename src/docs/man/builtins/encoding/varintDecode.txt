NAME
  varintDecode - decode a ZigZag varint

SYNOPSIS
  encoding::varintDecode(data AS List OF Byte) AS Integer

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes unsigned LEB128 and reverses the ZigZag mapping back to a signed Integer. Fails on malformed data.

PARAMETERS
  data AS List OF Byte
    The varint bytes.

RETURN VALUE
  AS Integer
    The decoded value.

ERRORS
  77050003 (ErrInvalidFormat)
  On overflow or truncation.

EXAMPLES
  IMPORT encoding

  encoding::varintDecode(bytes)

SEE ALSO
  mfb man encoding, mfb man bits
