NAME
  money::setRounding - set the current Money arithmetic rounding mode

SYNOPSIS
  money::setRounding(mode AS Rounding)

PACKAGE
  money

IMPORTS
  IMPORT money

DESCRIPTION
  money::setRounding selects how Money arithmetic settles the half case for the
  current execution context. The mode is one of the Rounding enum members:
  Commercial (round half away from zero, the default) or Banker (round half to
  even). It applies to every Money rounding site — division by a scalar, scaling
  by a Float or Fixed, money::round, and the toMoney/toFixed conversions.

  The mode is per-thread state. A worker thread inherits the spawning thread's
  mode at spawn and then changes independently. Setting the mode has no effect on
  Fixed or Float rounding, and does not change how toString(Money) renders a
  value: presentation rounding is a fixed half-away-from-zero rule independent of
  the mode.

PARAMETERS
  mode AS Rounding
    money::Rounding.Commercial or money::Rounding.Banker.

RETURN VALUE
  Nothing.

ERRORS
  None.

TYPE CHECKING
  The argument must be a money::Rounding value. Any other type is a compile-time
  error.

EXAMPLES
  Switch to banker's rounding for an accumulation, then restore the default:

    money::setRounding(Rounding.Banker)
    ' ... Money arithmetic here rounds half-to-even ...
    money::setRounding(Rounding.Commercial)

SEE ALSO
  money::getRounding, money::round, toMoney
