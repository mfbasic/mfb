NAME
  do - DO loop forms

SYNOPSIS
  DO
    ...
  LOOP UNTIL condition

  DO WHILE condition
    ...
  LOOP

DESCRIPTION
  DO supports two loop forms: a post-test `DO ... LOOP UNTIL condition`
  and a pre-test `DO WHILE condition ... LOOP`.

RULES
  `DO ... LOOP UNTIL condition` runs the body first, then stops when the
  Boolean condition becomes TRUE.

  `DO WHILE condition ... LOOP` checks the Boolean condition before each
  iteration and may run zero times.

  A bare `DO ... LOOP` form is not valid syntax.

EXAMPLES
  DO : work = work + 1 : LOOP UNTIL done
  DO WHILE ready : io::print("tick") : ready = FALSE : LOOP
