NAME
  formUrlEncode - form-urlencode text

SYNOPSIS
  encoding::formUrlEncode(text AS String) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Encodes text as application/x-www-form-urlencoded: spaces become +, and other non-alphanumeric characters are percent-encoded. Total.

PARAMETERS
  text AS String
    The text to encode.

RETURN VALUE
  AS String
    The form-encoded text.

ERRORS
  None.

EXAMPLES
  IMPORT encoding

  encoding::formUrlEncode("a b")
