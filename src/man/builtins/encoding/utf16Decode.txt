NAME
  utf16Decode - decode UTF-16 code units to a String

SYNOPSIS
  encoding::utf16Decode(value AS List OF Integer) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes a UTF-16 code-unit sequence to a String, combining surrogate pairs. Fails on an element outside 0..65535 or an unpaired surrogate.

PARAMETERS
  value AS List OF Integer
    The UTF-16 code units.

RETURN VALUE
  AS String
    The decoded text.

ERRORS
  77050003 (ErrInvalidFormat)
  On an out-of-range unit or unpaired surrogate.

EXAMPLES
  IMPORT encoding

  encoding::utf16Decode([104, 105])

  mfb man encoding, mfb man bits
