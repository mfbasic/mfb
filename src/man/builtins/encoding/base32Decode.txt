NAME
  base32Decode - decode Base32 to bytes

SYNOPSIS
  encoding::base32Decode(text AS String) AS List OF Byte

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes a Base32 string (case-insensitive) to bytes. Fails on invalid characters or invalid alignment padding.

PARAMETERS
  text AS String
    The Base32 string.

RETURN VALUE
  AS List OF Byte
    The decoded bytes.

ERRORS
  77050003 (ErrInvalidFormat)
  On invalid characters or padding.

EXAMPLES
  IMPORT encoding

  encoding::base32Decode("MZXW6===")

SEE ALSO
  mfb man encoding, mfb man bits
