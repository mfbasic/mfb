NAME
  base32Encode - encode bytes to Base32

SYNOPSIS
  encoding::base32Encode(data AS List OF Byte) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Encodes data to standard Base32 (RFC 4648 6), uppercase with '=' padding. Total.

PARAMETERS
  data AS List OF Byte
    The bytes to encode.

RETURN VALUE
  AS String
    The Base32 string.

ERRORS
  None.

EXAMPLES
  IMPORT encoding

  encoding::base32Encode(raw)

SEE ALSO
  mfb man encoding, mfb man bits
