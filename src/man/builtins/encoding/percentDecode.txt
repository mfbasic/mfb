NAME
  percentDecode - decode percent-encoded text

SYNOPSIS
  encoding::percentDecode(text AS String) AS String

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes percent-encoded text, interpreting the decoded bytes as UTF-8. Fails on a malformed %XX escape or invalid UTF-8.

PARAMETERS
  text AS String
    The percent-encoded text.

RETURN VALUE
  AS String
    The decoded text.

ERRORS
  77050003 (ErrInvalidFormat)
  On a malformed escape or bad UTF-8.

EXAMPLES
  IMPORT encoding

  encoding::percentDecode("a%20b")

  mfb man encoding, mfb man bits
