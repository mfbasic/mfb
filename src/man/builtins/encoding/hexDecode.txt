NAME
  hexDecode - decode hex to bytes

SYNOPSIS
  encoding::hexDecode(text AS String) AS List OF Byte

PACKAGE
  encoding

IMPORTS
  IMPORT encoding

DESCRIPTION
  Decodes hexadecimal text (upper or lower case) to bytes. Fails on a non-hex character or an odd length.

PARAMETERS
  text AS String
    The hex string.

RETURN VALUE
  AS List OF Byte
    The decoded bytes.

ERRORS
  77050003 (ErrInvalidFormat)
  On a non-hex character or odd length.

EXAMPLES
  IMPORT encoding

  encoding::hexDecode("68656c6c6f")

SEE ALSO
  mfb man encoding, mfb man bits
