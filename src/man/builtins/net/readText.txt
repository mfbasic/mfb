NAME
  readText - read available bytes from a connected socket as UTF-8 text

SYNOPSIS
  net::readText(sock AS Socket, maxBytes AS Integer) AS String

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  readText receives data from a connected Socket and returns it decoded as a
  UTF-8 String. At most maxBytes bytes are read in a single call; the result is
  often shorter because readText returns as soon as any data is available rather
  than waiting to fill the requested size. The socket must still be open.

  readText blocks until at least one byte arrives, the peer closes the
  connection, or the socket's read timeout elapses. An empty String signals end
  of stream: the peer has closed its side and no further data will arrive. Use
  net::poll to test for readiness without blocking, or net::setReadTimeout to
  bound how long a read may wait.

  maxBytes must be positive and caps the number of bytes read before decoding.
  Use net::read instead when the peer sends raw binary data, or when a single
  read may split a multi-byte UTF-8 sequence across calls and you need to
  reassemble the bytes before decoding.

ERRORS
  Raises an Error with code 77050002 (ErrInvalidArgument) when maxBytes is not
  positive, 77030004 (ErrResourceClosed) when the socket has already been closed,
  or 77070003 (ErrNetworkFailed) when the read could not be completed or the read
  timeout elapsed before any data arrived. Raises an Error when the received
  bytes are not valid UTF-8.

EXAMPLES
  Read up to 1024 bytes of text from a connected socket:

    IMPORT net

    RES client = net::connectTcp("example.com", 80)
    net::writeText(client, "GET / HTTP/1.0\r\n\r\n")
    LET reply = net::readText(client, 1024)
    ' client is closed by lexical drop when this scope ends

  Drain a connection until the peer closes it:

    IMPORT net

    RES client = net::connectTcp("example.com", 80)
    net::writeText(client, "GET / HTTP/1.0\r\n\r\n")
    MUT response AS String = ""
    DO
      LET chunk = net::readText(client, 4096)
      IF len(chunk) = 0 THEN
        EXIT DO
      END IF
      response = response & chunk
    LOOP
    ' client is closed by lexical drop when this scope ends
