NAME
  close - close a network resource and release its OS handle

SYNOPSIS
  net::close(sock AS Socket) AS Nothing
  net::close(listener AS Listener) AS Nothing
  net::close(sock AS UdpSocket) AS Nothing

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  close shuts down a network resource and releases the underlying OS socket. It
  accepts a connected Socket, a TCP Listener, or a UDP UdpSocket. After close
  returns, the resource must not be used again; further operations on it raise an
  Error.

  Closing a Socket or UdpSocket tears down the connection or binding, so a peer
  reading from a closed connection observes the end of the stream. Closing a
  Listener stops accepting new connections but does not affect Sockets already
  returned by net::accept; each of those is an independent resource closed on its
  own.

  Closing is otherwise automatic: every network resource is closed by lexical
  drop when its scope ends. Call net::close only when the handle must be released
  before then, such as freeing a listening port or a connection inside a
  long-running loop.

OVERLOADS
  net::close(sock AS Socket) AS Nothing
    Closes a connected TCP socket.

  net::close(listener AS Listener) AS Nothing
    Closes a TCP listener, stopping further accepts.

  net::close(sock AS UdpSocket) AS Nothing
    Closes a bound UDP socket.

ERRORS
  Raises an Error with code 77030004 (ErrResourceClosed) when the resource has
  already been closed, or 77070003 (ErrNetworkFailed) when the host OS reports a
  failure while releasing the handle.

EXAMPLES
  Release a listening port as soon as it is no longer needed:

    IMPORT net

    RES listener = net::listenTcp("0.0.0.0", 8080)
    RES client = net::accept(listener)
    net::close(listener)
    LET request = net::readText(client, 1024)
    ' client is closed by lexical drop when this scope ends

  Close a connection explicitly inside a loop:

    IMPORT net

    RES server = net::listenTcp("127.0.0.1", 8080)
    DO
      RES peer = net::accept(server)
      net::writeText(peer, "hello\n")
      net::close(peer)
    LOOP
