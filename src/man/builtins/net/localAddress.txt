NAME
  localAddress - report the local endpoint bound to a network resource

SYNOPSIS
  net::localAddress(sock AS Socket) AS Address
  net::localAddress(listener AS Listener) AS Address
  net::localAddress(sock AS UdpSocket) AS Address

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  localAddress returns the local endpoint that the host OS has bound to a network
  resource as an Address. It accepts a connected Socket, a TCP Listener, or a
  bound UDP UdpSocket. The Address reports the address this side of the resource
  occupies, such as the port a Listener is accepting on or the local port the OS
  assigned to an outbound connection.

  Each Address has a host field (String) and a port field (Integer). For a
  Listener bound to a wildcard host the reported host is the wildcard address; the
  port is the concrete port chosen by the OS, which is useful for discovering the
  port assigned when listening on port 0.

  Use net::remoteAddress to inspect the peer endpoint of a connected Socket
  instead of the local one.

OVERLOADS
  net::localAddress(sock AS Socket) AS Address
    Reports the local endpoint of a connected TCP socket.

  net::localAddress(listener AS Listener) AS Address
    Reports the endpoint a TCP listener is accepting on.

  net::localAddress(sock AS UdpSocket) AS Address
    Reports the endpoint a UDP socket is bound to.

ERRORS
  77030004 (ErrResourceClosed)
  Raised when the resource has already been closed, or when the host OS reports
  that it is no longer usable while querying its local endpoint.

  77070001 (ErrAddressInvalid)
  Raised when the local address reported by the OS cannot be represented as an
  Address.

  77010001 (ErrOutOfMemory)
  Raised when the resulting Address could not be allocated.

EXAMPLES
  Discover the port assigned when listening on port 0:

    IMPORT net

    RES listener = net::listenTcp("127.0.0.1", 0)
    LET local = net::localAddress(listener)
    LET port = local.port
    ' port holds the concrete port the OS chose for the listener
    ' listener is closed by lexical drop when this scope ends

  Inspect the local endpoint of an outbound connection:

    IMPORT net

    RES client = net::connectTcp("example.com", 80)
    LET local = net::localAddress(client)
    LET host = local.host
    LET port = local.port
    ' client is closed by lexical drop when this scope ends
