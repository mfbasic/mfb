NAME
  remoteAddress - report the peer endpoint of a connected network socket

SYNOPSIS
  net::remoteAddress(sock AS Socket) AS Address

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  remoteAddress returns the remote endpoint that a connected TCP Socket is
  communicating with as an Address. It reports the peer side of the connection,
  such as the host and port of the server an outbound connection reached or the
  client that an accepted connection originated from.

  Each Address has a host field (String) and a port field (Integer). The host is
  the concrete peer address resolved by the host OS, which may differ from any
  hostname passed to net::connectTcp once name resolution has occurred.

  Use net::localAddress to inspect the local endpoint of a Socket, Listener, or
  UdpSocket instead of the peer one.

ERRORS
  Raises an Error with code 77030004 (ErrResourceClosed) when the socket has
  already been closed or the host OS reports it is no longer usable, 77070001
  (ErrAddressInvalid) when the peer address cannot be represented as an Address,
  or 77010001 (ErrOutOfMemory) when the result could not be allocated.

EXAMPLES
  Inspect the peer endpoint of an outbound connection:

    IMPORT net

    RES client = net::connectTcp("example.com", 80)
    LET remote = net::remoteAddress(client)
    LET host = remote.host
    LET port = remote.port
    ' host and port hold the resolved peer the OS connected to
    ' client is closed by lexical drop when this scope ends

  Identify the client of an accepted connection:

    IMPORT net

    RES listener = net::listenTcp("127.0.0.1", 0)
    RES conn = net::accept(listener)
    LET remote = net::remoteAddress(conn)
    LET peerHost = remote.host
    ' peerHost holds the address the connecting client came from
    ' conn and listener are closed by lexical drop when this scope ends
