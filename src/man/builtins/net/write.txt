NAME
  write - write bytes to a connected socket

SYNOPSIS
  net::write(sock AS Socket, bytes AS List OF Byte) AS Nothing

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  write sends the raw bytes in bytes over a connected Socket. It writes the
  entire list before returning, looping internally to resend any portion that a
  single underlying write did not accept, so a successful call has delivered
  every byte to the socket's send buffer. The socket must still be open.

  An empty list writes nothing and returns immediately. Otherwise the call
  blocks while the send buffer is full, waiting until space becomes available or
  the socket's write timeout elapses. Bytes are sent in list order; the function
  does not allocate, reorder, or close the socket, and has no side effect beyond
  transmitting the data.

  Use net::setWriteTimeout to bound how long a write may wait, and
  net::writeText instead when sending UTF-8 text from a String is more
  convenient than building a List OF Byte.

PARAMETERS
  sock AS Socket
    A connected socket to send on, such as one returned by net::connectTcp or
    net::accept. It must still be open; writing to a closed socket is an error.

  bytes AS List OF Byte
    The bytes to send, transmitted in list order. An empty list writes nothing
    and returns immediately.

RETURN VALUE
  AS Nothing
    Returns nothing on success. A successful call has handed every byte in bytes
    to the socket's send buffer.

ERRORS
  77030004 (ErrResourceClosed)
  Raised when sock has already been closed.

  77070004 (ErrConnectionClosed)
  Raised when the peer has closed the connection, or the write otherwise fails
  for a reason other than a timeout.

  77070006 (ErrWriteTimeout)
  Raised when the socket's write timeout elapses before all bytes could be sent.

EXAMPLES
  Send a request as raw bytes over a connected socket:

    IMPORT net

    RES client = net::connectTcp("example.com", 80)
    LET request = [71, 69, 84, 32, 47, 13, 10, 13, 10] AS List OF Byte
    net::write(client, request)
    LET reply = net::read(client, 1024)
    ' client is closed by lexical drop when this scope ends

  Echo bytes received from a peer back to it:

    IMPORT net

    RES server = net::listenTcp("127.0.0.1", 8080)
    RES peer = net::accept(server)
    DO
      LET chunk = net::read(peer, 4096)
      IF len(chunk) = 0 THEN
        EXIT DO
      END IF
      net::write(peer, chunk)
    LOOP
    ' peer and server are closed by lexical drop when this scope ends
