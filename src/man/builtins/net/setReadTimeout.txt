NAME
  setReadTimeout - bound how long a read on a socket may block

SYNOPSIS
  net::setReadTimeout(sock AS Socket, timeoutMs AS Integer) AS Nothing
  net::setReadTimeout(sock AS UdpSocket, timeoutMs AS Integer) AS Nothing

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  setReadTimeout sets the maximum time, in milliseconds, that a receive on sock
  may block waiting for data. It applies to a connected TCP Socket or a bound UDP
  UdpSocket and takes effect on every subsequent read: net::read and
  net::readText for a Socket, and net::receiveFrom and net::receiveTextFrom for a
  UdpSocket.

  When the timeout elapses before any data arrives, the pending read fails with
  an Error rather than blocking further. The timeout governs only how long a read
  waits for the first data to become available; it does not cap the total time a
  loop of reads may take.

  A timeoutMs of 0 disables the timeout, so reads block indefinitely until data
  arrives, the peer closes the connection, or an error occurs. This is the
  default state of a freshly opened socket. timeoutMs must not be negative.

OVERLOADS
  net::setReadTimeout(sock AS Socket, timeoutMs AS Integer) AS Nothing
    Bounds reads on a connected TCP socket.

  net::setReadTimeout(sock AS UdpSocket, timeoutMs AS Integer) AS Nothing
    Bounds receives on a bound UDP socket.

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when timeoutMs is negative.

  77030004 (ErrResourceClosed)
  Raised when sock has already been closed, or when the timeout could not be
  applied to the underlying socket handle (the setsockopt call failed).

EXAMPLES
  Fail a read that stalls for more than two seconds:

    IMPORT net

    RES client = net::connectTcp("example.com", 80)
    net::setReadTimeout(client, 2000)
    net::writeText(client, "GET / HTTP/1.0\r\n\r\n")
    LET chunk = net::read(client, 4096)
    ' client is closed by lexical drop when this scope ends

  Bound a UDP receive so a missing reply does not block forever:

    IMPORT net

    RES sock = net::bindUdp("0.0.0.0", 0)
    net::setReadTimeout(sock, 1000)
    LET datagram = net::receiveTextFrom(sock, 1024)
    ' sock is closed by lexical drop when this scope ends
