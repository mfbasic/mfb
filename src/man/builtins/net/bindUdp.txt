NAME
  bindUdp - open a UDP socket bound to a local address

SYNOPSIS
  net::bindUdp(host AS String, port AS Integer) AS UdpSocket

PACKAGE
  net

IMPORTS
  IMPORT net

DESCRIPTION
  bindUdp binds a local UDP socket to host and port and returns a UdpSocket
  resource ready to send and receive datagrams. host names the local interface
  to bind; use "0.0.0.0" or "::" to receive on any interface. When port is 0 the
  host assigns an ephemeral port, which can be read back with net::localAddress.

  Unlike TCP, a UDP socket is connectionless: there is no listen or accept step.
  Send datagrams to a peer with net::sendTo or net::sendTextTo, and receive them
  with net::receiveFrom or net::receiveTextFrom, each of which reports the sender
  address alongside the payload.

  UdpSocket resources are closed by lexical drop at scope exit or explicitly with
  net::close, and the bound endpoint can be inspected with net::localAddress.

ERRORS
  Raises an Error with code 77050002 (ErrInvalidArgument) when an argument value
  is not valid for the operation, 77070001 (ErrAddressInvalid) when the host or
  port is malformed, or 77070003 (ErrNetworkFailed) when the socket could not be
  bound, such as when the address is already in use.

EXAMPLES
  Bind on all interfaces and receive a datagram:

    IMPORT net

    RES sock = net::bindUdp("0.0.0.0", 9000)
    LET datagram = net::receiveTextFrom(sock, 1024)
    ' datagram.value holds the text; datagram.address holds the sender
    ' sock is closed by lexical drop when this scope ends

  Bind an ephemeral port and read back the assigned address:

    IMPORT net

    RES sock = net::bindUdp("127.0.0.1", 0)
    LET bound = net::localAddress(sock)
    ' bound.port holds the host-assigned port
