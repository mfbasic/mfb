NAME
  bswap64 - reverse the byte order of all 64 bits

SYNOPSIS
  bits::bswap64(value AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Reverses the order of the eight bytes that make up the 64-bit value and returns
  the result. Byte 0 becomes byte 7, byte 1 becomes byte 6, byte 2 becomes byte 5,
  byte 3 becomes byte 4, and so on, so that the most significant byte and the least
  significant byte are exchanged and every byte in between is mirrored. This swaps
  the value between little-endian and big-endian byte order.

  The operand is treated as a raw two's-complement bit pattern; the function does
  not interpret its sign except as the Integer signature states. The operation is
  total: it is defined for every Integer value and never fails. The result uses all
  64 bits, so no bits are cleared.

PARAMETERS
  value AS Integer
    The value whose eight bytes are reversed. All 64 bits participate in the swap.

RETURN VALUE
  AS Integer
    The fully byte-reversed value, with byte 0 and byte 7 exchanged, byte 1 and
    byte 6 exchanged, byte 2 and byte 5 exchanged, and byte 3 and byte 4 exchanged.

ERRORS
  No errors.

EXAMPLES
  Swap the eight bytes of a 64-bit value:

    IMPORT bits

    LET result AS Integer = bits::bswap64(255)
