NAME
  bnot - bitwise NOT

SYNOPSIS
  bits::bnot(a AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Returns the one's complement of a: every one of the 64 bits is inverted, so
  bit i of the result is 1 exactly when bit i of a is 0. As a two's-complement
  arithmetic identity this equals -(a) - 1.

  The operand is a raw two's-complement 64-bit Integer bit pattern; the function
  does not interpret sign. The operation is total, has no side effects, and lowers
  to a single native AArch64 instruction inline, producing identical results on the
  native and Binary Representation execution paths.

PARAMETERS
  a AS Integer
    The operand to invert. Any 64-bit value.

RETURN VALUE
  AS Integer
    The bitwise complement of a, with all 64 bits inverted.

ERRORS
  No errors.

EXAMPLES
  Invert every bit of a value:

    IMPORT bits

    LET result AS Integer = bits::bnot(255)
