NAME
  popCount - count set bits (population count)

SYNOPSIS
  bits::popCount(value AS Integer) AS Integer

PACKAGE
  bits

IMPORTS
  IMPORT bits

DESCRIPTION
  Returns the number of set (1) bits in the 64-bit value, also known as its
  Hamming weight or population count. The value is treated as a raw two's-
  complement 64-bit bit pattern; the sign is not interpreted, so every one of
  the 64 bit positions is inspected regardless of whether value is negative.
  When value is 0 no bits are set and the result is 0; when every bit is set
  (the bit pattern -1) the result is 64. This operation is total: it is defined
  for every Integer and never fails. It lowers to a single native AArch64
  population-count instruction.

PARAMETERS
  value AS Integer
    The 64-bit value to inspect. Any Integer is accepted.

RETURN VALUE
  AS Integer
    The count of set bits, in the range 0..64. The result is 0 when value is 0
    and 64 when every bit is set.

ERRORS
  No errors.

EXAMPLES
  Count the set bits of a small value:

    IMPORT bits

    LET result AS Integer = bits::popCount(255)
