NAME
  get - read a nested JSON object value by key path

SYNOPSIS
  json::get(value AS Json, path AS List OF String) AS Json

PACKAGE
  json

DESCRIPTION
  json::get walks path through nested Json object members and returns the value
  found there. Each path element is matched against an object key. An empty path
  returns value unchanged.

  The path argument may also be passed by the name key.

ERRORS
  ErrNotFound (77050004)
    Returned when any path element is absent or when traversal reaches a Json
    value that is not an object before the path is exhausted.

EXAMPLES
  LET city AS Json = json::get(doc, ["user", "address", "city"])

  LET root AS Json = json::get(doc, [])
