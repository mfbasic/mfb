NAME
  set - return a collection with one value changed

SYNOPSIS
  collections::set OF T(value AS List OF T, index AS Integer, item AS T) AS List OF T
  collections::set OF K, V(value AS Map OF K TO V, key AS K, item AS V) AS Map OF K TO V

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  set returns a new collection with one value changed. The original collection
  is not mutated.

  For a List, set replaces the item at the given zero-based index. The list keeps
  its length; the index must refer to an existing element. For a Map, set returns
  a map with key associated with item, adding the key when it is not already
  present and replacing the value when it is.

OVERLOADS
  set OF T(value AS List OF T, index AS Integer, item AS T) AS List OF T
    Returns a list with item at index.

  set OF K, V(value AS Map OF K TO V, key AS K, item AS V) AS Map OF K TO V
    Returns a map with key set to item.

ERRORS
  77050001
    Returned by the List overload when index is out of range. The Map overload
    does not fail for supported argument types.

  Other errors can propagate from evaluating arguments before set is called.

TYPE CHECKING
  List set requires an Integer index and an item of the list element type. Map
  set requires key and item to match the map key and value types.

EXAMPLES
  Replace a list item:

    LET numbers AS List OF Integer = set([1, 2, 3], 1, 9)

  Set a map value:

    LET scores AS Map OF String TO Integer = set(Map OF String TO Integer {}, "Ada", 10)

SEE ALSO
