NAME
  forEach - run an action for each list item

SYNOPSIS
  collections::forEach OF T(value AS List OF T, action AS FUNC(T) AS Nothing) AS Nothing

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  forEach calls action once for each item in value, in order from left to right.
  The list is not copied or mutated.

  A SUB(T) is accepted for action because a SUB has success type Nothing. If
  action fails for any item, the error auto-propagates and iteration stops, so
  later items are not visited.

OVERLOADS
  forEach OF T(value AS List OF T, action AS FUNC(T) AS Nothing) AS Nothing
    Runs action for every item and returns Nothing.

ERRORS
  forEach propagates errors raised by action, stopping at the failing item.
  Errors can still propagate from evaluating value or action before forEach is
  called.

TYPE CHECKING
  The first argument must be a List OF T. The second argument must accept one
  argument of the list element type T and return Nothing. Calling forEach on a
  non-list value, or with an action whose parameter type does not match T, is a
  compile-time type error.

EXAMPLES
  Print each string:

    forEach(["hello", "world"], io::print)

  Use a local action:

    SUB printNumber(value AS Integer)
      io::print(toString(value))
    END SUB

    forEach([1, 2, 3], printNumber)
