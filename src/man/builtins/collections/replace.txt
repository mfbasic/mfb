NAME
  replace - replace matching list items

SYNOPSIS
  collections::replace OF T(value AS List OF T, old AS T, new AS T) AS List OF T

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  replace returns a new list where every item equal to old is replaced with
  new. The original list is not mutated.

OVERLOADS
  replace OF T(value AS List OF T, old AS T, new AS T) AS List OF T
    Replaces every matching list item.

ERRORS
  replace does not fail for supported evaluated arguments. Errors can still
  propagate from evaluating arguments before replace is called.

TYPE CHECKING
  old and new must have the list element type T.

EXAMPLES
  Replace list items:

    LET values AS List OF Integer = replace([1, 2, 1], 1, 9)
