NAME
  sum - add the values in a numeric list

SYNOPSIS
  collections::sum(value AS List OF Integer) AS Integer
  collections::sum(value AS List OF Float) AS Float
  collections::sum(value AS List OF Fixed) AS Fixed

PACKAGE
  collections

IMPORTS
  IMPORT collections

DESCRIPTION
  sum returns the total of a numeric list. It supports Integer, Float, and Fixed
  element types.

  For empty lists, the result is the zero value for the element type.

OVERLOADS
  sum(value AS List OF Integer) AS Integer
    Sums Integer values.

  sum(value AS List OF Float) AS Float
    Sums Float values.

  sum(value AS List OF Fixed) AS Fixed
    Sums Fixed values.

ERRORS
  ErrOverflow (77050010)
    Returned when Fixed summation overflows. Integer arithmetic in MFBASIC is
    checked; an Integer summation that exceeds the Integer range fails with the
    same overflow code.

  Other errors can propagate from evaluating value before sum is called.

TYPE CHECKING
  sum accepts only List OF Integer, List OF Float, and List OF Fixed.

EXAMPLES
  Sum integers:

    LET total AS Integer = sum([1, 2, 3])

  Sum fixed-point values:

    LET values AS List OF Fixed = [toFixed("1.25"), toFixed("2.50")]
    LET total AS Fixed = sum(values)
