NAME
  removeAt - remove one list item by index

SYNOPSIS
  removeAt OF T(value AS List OF T, index AS Integer) AS List OF T

PACKAGE
  collection

IMPORTS
  None. collection functions are available without an IMPORT statement.

DESCRIPTION
  removeAt returns a new list with the item at index removed. The original list is
  not mutated.

  index is zero-based. Valid indices run from 0 through the list length minus one.
  Removing index 0 drops the first item, and removing the last valid index drops the
  final item. The returned list is one item shorter than value.

OVERLOADS
  removeAt OF T(value AS List OF T, index AS Integer) AS List OF T
    Returns a list without the item at index.

ERRORS
  77050001
    Returned when index is out of range (less than 0 or greater than or equal to the
    list length).

  Other errors can propagate from evaluating arguments before removeAt is called.

TYPE CHECKING
  value must be a List OF T and index must be Integer. The result has the same list
  type as value.

EXAMPLES
  Remove the second item:

    LET numbers AS List OF Integer = removeAt([1, 2, 3], 1)
