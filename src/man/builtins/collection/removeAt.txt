NAME
  removeAt - remove one list item by index

SYNOPSIS
  removeAt OF T(value AS List OF T, index AS Integer) AS List OF T

PACKAGE
  collection

IMPORTS
  None. collection functions are available without an IMPORT statement.

DESCRIPTION
  removeAt returns a new list without the item at zero-based index. The original
  list is not mutated.

OVERLOADS
  removeAt OF T(value AS List OF T, index AS Integer) AS List OF T
    Returns a list without the item at index.

ERRORS
  10001
    Returned when index is out of range.

  Other errors can propagate from evaluating arguments before removeAt is called.

TYPE CHECKING
  removeAt accepts a List OF T and an Integer index.

EXAMPLES
  Remove the second item:

    LET numbers AS List OF Integer = removeAt([1, 2, 3], 1)

SEE ALSO
  insert
  removeKey
  set
