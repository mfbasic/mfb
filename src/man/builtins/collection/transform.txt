NAME
  transform - map each list item to a new value

SYNOPSIS
  transform OF T, U(value AS List OF T, f AS FUNC(T) AS U) AS List OF U

PACKAGE
  collection

IMPORTS
  None. collection functions are available without an IMPORT statement.

DESCRIPTION
  transform calls f once for each item in value, from left to right, and returns
  a new list containing each result. The original list is not mutated.

  If f fails for any item, the error auto-propagates and no successful result
  list is returned.

OVERLOADS
  transform OF T, U(value AS List OF T, f AS FUNC(T) AS U) AS List OF U
    Maps every item through f.

ERRORS
  transform propagates errors raised by f. Errors can also propagate from
  evaluating value or f before transform is called.

TYPE CHECKING
  f must accept one argument of the list element type T. The result list element
  type is f's success type U.

EXAMPLES
  Convert numbers to strings:

    LET text AS List OF String = transform([1, 2], toString)

  Use a local function:

    FUNC double(value AS Integer) AS Integer
      RETURN value * 2
    END FUNC

    LET doubled AS List OF Integer = transform([1, 2, 3], double)

SEE ALSO

  filter
  reduce
