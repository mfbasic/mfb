NAME
  reduce - fold a list into one accumulated value

SYNOPSIS
  reduce OF T, U(value AS List OF T, initial AS U, f AS FUNC(U, T) AS U) AS U

PACKAGE
  collection

IMPORTS
  None. collection functions are available without an IMPORT statement.

DESCRIPTION
  reduce folds value from left to right. The accumulator starts as initial. For
  each item, reduce calls f(accumulator, item) and uses the returned value as the
  next accumulator. The final accumulator is returned.

  If value is empty, reduce returns initial without calling f.

OVERLOADS
  reduce OF T, U(value AS List OF T, initial AS U, f AS FUNC(U, T) AS U) AS U
    Folds items left to right.

ERRORS
  reduce propagates errors raised by f. Errors can also propagate from
  evaluating value, initial, or f before reduce is called.

TYPE CHECKING
  The first argument must be a List OF T. The second argument fixes the
  accumulator type U. The third argument must be an f that accepts the
  accumulator type U followed by the list element type T and returns U. The
  result type is U. Calling reduce on a non-list value, with an initial whose
  type does not match f's first parameter and return type, or with an f whose
  parameter types do not match U and T, is a compile-time type error.

EXAMPLES
  Sum with a reducer:

    FUNC add(total AS Integer, value AS Integer) AS Integer
      RETURN total + value
    END FUNC

    LET total AS Integer = reduce([1, 2, 3], 0, add)

  Build text:

    FUNC joinWords(text AS String, word AS String) AS String
      IF text = "" THEN RETURN word
      RETURN text & " " & word
    END FUNC

    LET sentence AS String = reduce(["hello", "world"], "", joinWords)
