NAME
  append - return a list with an item or items added at the end

SYNOPSIS
  append OF T(value AS List OF T, item AS T) AS List OF T
  append OF T(value AS List OF T, items AS List OF T) AS List OF T

PACKAGE
  collection

IMPORTS
  None. collection functions are available without an IMPORT statement.

DESCRIPTION
  append returns a new list with content added at the end. The original list is
  not mutated.

  Given a single item, append returns a list one element longer with item placed
  after the existing items. Given a list of items, append returns a list with all
  of those items added in order after the existing items; appending an empty list
  returns a copy of the original.

OVERLOADS
  append OF T(value AS List OF T, item AS T) AS List OF T
    Returns a list with item added after the existing items.

  append OF T(value AS List OF T, items AS List OF T) AS List OF T
    Returns a list with all items added after the existing items.

ERRORS
  append does not fail for supported argument types. Errors can still propagate
  from evaluating arguments before append is called.

TYPE CHECKING
  The first argument must be a List OF T. The second argument must be either an
  item of the list element type T or another List OF T. Calling append on a
  non-list value, or with a mismatched element type, is a compile-time type
  error.

EXAMPLES
  Append one item:

    LET numbers AS List OF Integer = append([1, 2], 3)

  Append many items:

    LET numbers AS List OF Integer = append([1, 2], [3, 4])

SEE ALSO
  prepend, insert, set, removeAt, len
