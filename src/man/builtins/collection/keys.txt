NAME
  keys - return the keys from a map

SYNOPSIS
  keys OF K, V(value AS Map OF K TO V) AS List OF K

PACKAGE
  collection

IMPORTS
  None. collection functions are available without an IMPORT statement.

DESCRIPTION
  keys returns the keys from value in implementation-defined stable order. The
  result list contains one item for each map entry.

OVERLOADS
  keys OF K, V(value AS Map OF K TO V) AS List OF K
    Returns the map keys.

ERRORS
  keys does not fail for an evaluated map. Errors can still propagate from
  evaluating value before keys is called.

TYPE CHECKING
  keys accepts only Map OF K TO V values.

EXAMPLES
  Get map keys:

    LET names AS List OF String = keys(Map OF String TO Integer { "a" := 1 })

SEE ALSO
  values
  hasKey
  len
