NAME
  header - look up a response header, case-insensitively

SYNOPSIS
  http::header(result AS Result, name AS String) AS String

PACKAGE
  http

IMPORTS
  IMPORT http

DESCRIPTION
  header returns the value of the response header named name from result. The match
  is case-insensitive (HTTP field names are case-insensitive), so header(r, "host")
  finds a field received as Host. When a field appears more than once, the first
  occurrence in received order is returned. Use http::headerOr to supply a default
  instead of failing when the field is absent.

ERRORS
  77050004 (ErrNotFound)
  Raised when no response header matches name.

EXAMPLES
    IMPORT http

    LET ct = http::header(r, "Content-Type")

SEE ALSO
  http::headerOr, http::read, http::write
