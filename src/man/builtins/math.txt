package: math
summary: Deterministic numeric helper functions.
usage: mfb man math [function]
---
function: pi
signature: math::pi AS Float
summary: The mathematical constant pi as a Float.
example:
  LET value AS Float = math::pi

function: abs
signature: math::abs(value AS Integer) AS Integer; math::abs(value AS Float) AS Float; math::abs(value AS Fixed) AS Fixed
summary: Returns the absolute value of a number. Integer and Fixed minimum-value overflow fails with 10028.
example:
  LET value AS Integer = math::abs(-4)
---
function: min
signature: math::min(a AS Integer, b AS Integer) AS Integer; math::min(a AS Float, b AS Float) AS Float; math::min(a AS Fixed, b AS Fixed) AS Fixed
summary: Returns the smaller of two numeric values of the same type.
example:
  LET value AS Integer = math::min(2, 4)
---
function: max
signature: math::max(a AS Integer, b AS Integer) AS Integer; math::max(a AS Float, b AS Float) AS Float; math::max(a AS Fixed, b AS Fixed) AS Fixed
summary: Returns the larger of two numeric values of the same type.
example:
  LET value AS Integer = math::max(2, 4)
---
function: clamp
signature: math::clamp(value AS Integer, low AS Integer, high AS Integer) AS Integer; math::clamp(value AS Float, low AS Float, high AS Float) AS Float; math::clamp(value AS Fixed, low AS Fixed, high AS Fixed) AS Fixed
summary: Restricts a numeric value to an inclusive range. Fails with 10002 when low is greater than high.
example:
  LET value AS Integer = math::clamp(12, 0, 10)
---
function: floor
signature: math::floor(value AS Float) AS Integer; math::floor(value AS Fixed) AS Integer
summary: Returns the greatest integer less than or equal to the value.

  LET value AS Integer = math::floor(3.75)
---
function: ceil
signature: math::ceil(value AS Float) AS Integer; math::ceil(value AS Fixed) AS Integer
summary: Returns the smallest integer greater than or equal to the value.

  LET value AS Integer = math::ceil(3.25)
---
function: round
signature: math::round(value AS Float) AS Integer; math::round(value AS Fixed) AS Integer
summary: Returns the nearest integer, with halves rounded away from zero.
