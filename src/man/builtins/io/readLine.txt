NAME
  readLine - read a line of text from standard input

SYNOPSIS
  io::readLine() AS String

PACKAGE
  io

IMPORTS
  IMPORT io

DESCRIPTION
  Reads a single line from standard input and returns it as a String. Bytes are
  read from file descriptor 0 one at a time until a newline (LF, byte 10) is
  seen or end of file is reached. The trailing newline is removed, and a
  trailing carriage return (from a CRLF line ending) is also stripped, so the
  returned String holds only the line's content.

  Unlike io::input, no prompt is written before reading. To prompt the user on
  the same line, write the prompt with io::write and flush it with io::flush
  first, or use io::input with a prompt argument instead.

  Input is decoded as UTF-8 as it is read. Each multi-byte sequence is validated
  and may encode any valid Unicode scalar value, including emoji; an ill-formed
  byte sequence fails rather than producing replacement characters. The input
  buffer grows as needed to hold the line, so there is no fixed length limit
  other than available memory.

  End of input is reported as an error, not as an empty result. A line that is
  empty before its newline (an immediate LF, or CRLF) returns an empty String
  while still consuming the terminator; only a read that reaches end of file
  before any byte of the line is read fails with end-of-input. Use io::pollInput
  to test whether input is available before calling.

  io::readLine reads from the same standard input stream as io::input — the only
  difference is the optional prompt — so the two should not be interleaved in a
  way that splits a single logical line. The function does not alter the terminal
  mode; typed characters are echoed by the terminal in the normal cooked-mode
  manner.

PARAMETERS
  None.

RETURN VALUE
  AS String
    The line read from standard input, with the trailing LF and any preceding CR
    removed. A line that is empty before its newline returns an empty String.

ERRORS
  77020003 (ErrEof)
  Raised when standard input reaches end of file before any byte of the line is
  read.

  77020004 (ErrEncoding)
  Raised when the bytes read do not form a valid UTF-8 sequence.

  77020005 (ErrInput)
  Raised when reading from standard input fails for any other reason.

  77010001 (ErrOutOfMemory)
  Raised when the line buffer or the returned String cannot be allocated.

EXAMPLES
  Read a line and echo it back:

    LET line AS String = io::readLine()
    io::print(line)

  Prompt on the same line by writing and flushing first:

    io::write("Name: ")
    io::flush()
    LET name AS String = io::readLine()
    io::print("Hello, " + name)

  Read a line only when input is available:

    IF io::pollInput() THEN
      io::print(io::readLine())
    END IF
