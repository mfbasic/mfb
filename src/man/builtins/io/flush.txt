NAME
  flush - flush standard output

SYNOPSIS
  io::flush() AS Nothing

DESCRIPTION
  Flushes standard output. The native runtime writes directly to the host
  standard-output stream and does not add an additional MFBASIC output buffer.

  Fails with ErrOutput (10015) on output failure.

EXAMPLES
  io::write("ready")
  io::flush()
