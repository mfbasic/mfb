NAME
  io - Terminal and standard stream input/output functions

SYNOPSIS
  IMPORT io
  io::print(value)
  io::write(value)

DESCRIPTION
  The io package provides standard input, standard output, standard error, and
  terminal inspection helpers.

  Output functions accept String values only. Use toString explicitly before
  printing non-string values.

  Thread cancellation is cooperative. The current runtime does not
  asynchronously interrupt arbitrary standard-input reads; workers that need
  prompt cancellation around input should use nonblocking or timed polling and
  check thread::isCancelled between waits.
