NAME
    normalizeNfc - normalize a string to Unicode NFC

SYNOPSIS
    strings::normalizeNfc(value AS String) AS String

DESCRIPTION
    Returns value normalized to Unicode Normalization Form C.

EXAMPLES
    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(strings::normalizeNfc("Café"))
      RETURN 0
    END FUNC

FIXME
  None.

Untested: Edge cases not explicitly covered include canonical combining-mark reordering and non-composing combining sequences that should remain valid NFC behavior.
