NAME
  find - locate the first occurrence of a substring by Unicode scalar index

SYNOPSIS
  strings::find(value AS String, needle AS String, start AS Integer = 0) AS Integer

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  find searches value for the first occurrence of needle at or after the
  position start and returns the zero-based index where it begins.

  Positions are measured in Unicode scalar values, not UTF-8 bytes and not
  user-perceived grapheme clusters. A multi-byte character such as "é" or "😀"
  counts as one position even though it occupies several bytes. Both the start
  argument and the returned index are scalar indexes.

  The search begins at scalar position start and scans toward the end of value.
  The first position whose following scalars match needle in full is returned.
  An empty needle matches immediately and returns start, provided start is a
  valid position in value.

  start defaults to 0, meaning the search begins at the start of value. start
  must be in the range 0 through the scalar length of value inclusive; the
  upper bound equals the length so that a search may begin at the end of the
  string (where only an empty needle can match). A negative start, or one
  greater than the scalar length, is out of range.

  find does not mutate value or needle and has no side effects. The returned
  Integer is a fresh value.

  find is also defined for collections; see collections find for the List
  overload.

OVERLOADS
  strings::find(value AS String, needle AS String, start AS Integer = 0) AS Integer
    Returns the first scalar index at or after start where needle occurs in
    value.

PARAMETERS
  value AS String
    The string to search.

  needle AS String
    The substring to locate. An empty needle matches at start.

  start AS Integer
    The zero-based scalar index at which to begin searching. Defaults to 0.
    Must be between 0 and the scalar length of value inclusive.

RETURN VALUE
  AS Integer
    The zero-based Unicode scalar index of the first occurrence of needle at or
    after start. When needle is empty, returns start.

ERRORS
  77050001 (ErrIndexOutOfRange)
  Raised when start is less than 0 or greater than the scalar length of value.

  77050004 (ErrNotFound)
  Raised when no occurrence of needle exists at or after start.

TYPE CHECKING
  value and needle must both be String. start must be an Integer.

EXAMPLES
  Find text:

    LET index AS Integer = strings::find("hello", "l")

  Start after an earlier match:

    LET index AS Integer = strings::find("hello", "l", 3)

  Handle absence explicitly:

    IMPORT strings
    IMPORT errorCode

    MATCH strings::find("abc", "z")
      CASE Ok(i) : io::print("found at " & toString(i))
      CASE Err(e) WHEN e.code = errorCode::ErrNotFound : io::print("not found")
      CASE Err(e) : FAIL e
    END MATCH
