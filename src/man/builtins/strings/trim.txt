NAME
  trim - remove leading and trailing Unicode whitespace

SYNOPSIS
  strings::trim(value AS String) AS String

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  trim returns a new String equal to value with all leading and trailing
  Unicode whitespace removed. Whitespace is determined by the Unicode
  White_Space property, so in addition to the ASCII space, tab, carriage
  return, and newline it also removes characters such as the no-break space,
  the ideographic space, and other Unicode space separators.

  Only whitespace at the very start and the very end of value is removed.
  Whitespace that appears between non-whitespace characters is left untouched,
  so the interior of the string, including any embedded spaces or line breaks,
  is preserved exactly.

  If value contains only whitespace, the result is the empty string. If value
  has no leading or trailing whitespace, a string equal to value is returned.
  trim reads value only and has no side effects; it never mutates its argument.
  To strip a specific set of scalars rather than whitespace, use
  strings::trimChars. To trim only one end, use strings::trimStart or
  strings::trimEnd.

PARAMETERS
  value AS String
    The string to strip. Any sequence of Unicode whitespace at the beginning
    and end is removed; interior content is preserved.

RETURN VALUE
  AS String
    A new string equal to value with leading and trailing Unicode whitespace
    removed. Returns the empty string when value is empty or consists entirely
    of whitespace, and a string equal to value when value has no surrounding
    whitespace.

ERRORS
  No errors.

EXAMPLES
  Remove surrounding spaces:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(strings::trim("  Hello  "))
      RETURN 0
    END FUNC

  Interior whitespace is preserved:

    LET cleaned AS String = strings::trim("\n  a b  \n")

SEE ALSO
  strings::trimStart, strings::trimEnd, strings::trimChars
