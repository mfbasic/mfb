NAME
    graphemes - split a string into extended grapheme clusters

SYNOPSIS
    strings::graphemes(value AS String) AS List OF String

DESCRIPTION
    Splits value into Unicode extended grapheme clusters.

EXAMPLES
    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      LET parts = strings::graphemes("a😀")
      io::print(toString(len(parts)))
      RETURN 0
    END FUNC

FIXME
  None.

Untested: Edge cases needing coverage include non-ASCII scalar values, combining marks, ZWJ emoji sequences, regional-indicator flags, CRLF/control boundaries, Hangul syllable clusters, spacing marks, prepend characters, and Indic conjunct/GB9c sequences.
