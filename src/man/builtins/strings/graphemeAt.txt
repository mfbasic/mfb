NAME
  graphemeAt - return the extended grapheme cluster at a grapheme index

SYNOPSIS
  strings::graphemeAt(value AS String, index AS Integer) AS String

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  graphemeAt returns the single extended grapheme cluster located at the
  zero-based grapheme index in value, as a new String.

  An extended grapheme cluster is one user-perceived character. It may consist
  of several Unicode scalar values: a base letter followed by combining marks,
  a flag built from a pair of regional indicators, or an emoji formed from a
  base symbol joined to modifiers and zero-width joiners. graphemeAt counts and
  indexes by these clusters, not by Unicode scalar values (as len does) or by
  UTF-8 bytes (as strings::byteLen does). Indexing is the same enumeration that
  strings::graphemes produces: graphemeAt(value, i) yields the element at
  position i of strings::graphemes(value).

  index is zero-based: 0 selects the first cluster, and the last valid index is
  one less than strings::graphemesCount(value). An index that is negative or
  greater than or equal to the grapheme count is out of range and raises an
  error; it is not clamped or wrapped. The empty string has no clusters, so
  every index is out of range for it.

  value is not modified and graphemeAt has no side effects; the returned String
  is a fresh value holding a copy of the selected cluster.

PARAMETERS
  value AS String
    The string to index. Its content is interpreted as Unicode text when
    determining cluster boundaries. Any String is accepted, including the empty
    string (for which all indices are out of range).

  index AS Integer
    The zero-based grapheme index to retrieve. Must satisfy
    0 <= index < strings::graphemesCount(value). Negative values and values at
    or beyond the grapheme count are out of range.

RETURN VALUE
  AS String
    A new String containing exactly the one extended grapheme cluster at the
    given index. The returned cluster may comprise multiple Unicode scalar
    values and multiple UTF-8 bytes.

ERRORS
  77050001 (ErrIndexOutOfRange)
  Raised when index is negative or greater than or equal to the number of
  grapheme clusters in value. Because the empty string has no clusters, any
  index applied to it is out of range.

EXAMPLES
  Retrieve a multi-scalar emoji by grapheme index:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(strings::graphemeAt("a😀b", 1))
      RETURN 0
    END FUNC

SEE ALSO
