NAME
  trimStart - remove leading Unicode whitespace

SYNOPSIS
  strings::trimStart(value AS String) AS String

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  trimStart returns a new String equal to value with all leading Unicode
  whitespace removed. Whitespace is determined by the Unicode White_Space
  property, so in addition to the ASCII space, tab, carriage return, and
  newline it also removes characters such as the no-break space, the
  ideographic space, and other Unicode space separators.

  Only whitespace at the very start of value is removed. Trailing whitespace
  and any whitespace that appears between non-whitespace characters are left
  untouched, so the end and interior of the string, including embedded spaces
  or line breaks, are preserved exactly.

  If value contains only whitespace, the result is the empty string. If value
  has no leading whitespace, a string equal to value is returned. trimStart
  reads value only and has no side effects; it never mutates its argument. To
  remove whitespace from the end instead, use strings::trimEnd, and to remove
  it from both ends, use strings::trim. To strip a specific set of scalars
  rather than whitespace, use strings::trimChars.

PARAMETERS
  value AS String
    The string to strip. Any sequence of Unicode whitespace at the beginning
    is removed; trailing and interior content is preserved.

RETURN VALUE
  AS String
    A new string equal to value with leading Unicode whitespace removed.
    Returns the empty string when value is empty or consists entirely of
    whitespace, and a string equal to value when value has no leading
    whitespace.

ERRORS
  No errors.

EXAMPLES
  Remove leading spaces:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(strings::trimStart("  Hello  "))
      RETURN 0
    END FUNC

  Trailing whitespace is preserved:

    LET cleaned AS String = strings::trimStart("\n  a b  \n")

SEE ALSO
  strings::trimEnd, strings::trim, strings::trimChars
