NAME
    lower - convert a string to lowercase

SYNOPSIS
    strings::lower(value AS String) AS String

DESCRIPTION
    Converts value to lowercase using Unicode case mapping.

EXAMPLES
    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(strings::lower("HELLO"))
      RETURN 0
    END FUNC

FIXME
  None.

Untested: non-ASCII uppercase input, Unicode lowercase mappings that expand or combine code points, already-lowercase non-ASCII text, and context-sensitive lowercase cases such as Greek sigma if those are expected by “Unicode case mapping”.
