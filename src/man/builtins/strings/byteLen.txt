NAME
  byteLen - return the UTF-8 byte length of a string

SYNOPSIS
  strings::byteLen(value AS String) AS Integer

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  byteLen returns the number of bytes required to encode value in its UTF-8
  representation. It measures storage size, not character count: each byte of
  the encoding is counted exactly once.

  Because UTF-8 uses a variable number of bytes per Unicode scalar value, the
  result can be larger than len(value), which counts scalar values. ASCII
  characters occupy one byte each, so for a pure-ASCII string the two counts are
  equal; characters outside ASCII occupy two to four bytes each, making the byte
  length larger. The empty string has a byte length of zero.

  The function inspects value only; it does not mutate it or allocate a new
  String, and it is locale-independent.

PARAMETERS
  value AS String
    The string to measure. Any String is accepted, including the empty string.

RETURN VALUE
  AS Integer
    The number of bytes in the UTF-8 encoding of value. Returns 0 for the empty
    string. The result is always greater than or equal to len(value), and equal
    to it when value contains only ASCII characters.

ERRORS
  No errors.

EXAMPLES
  An ASCII string has one byte per character:

    LET n AS Integer = strings::byteLen("Hi")

  A non-ASCII character takes more than one byte:

    LET emoji AS Integer = strings::byteLen("😀")

  Compare byte length with scalar count for non-ASCII text:

    IMPORT io
    IMPORT strings

    FUNC main AS Integer
      io::print(toString(strings::byteLen("café")))
      io::print(toString(len("café")))
      RETURN 0
    END FUNC
