NAME
  left - return the leading Unicode scalars of a string

SYNOPSIS
  strings::left(value AS String, count AS Integer) AS String

PACKAGE
  strings

IMPORTS
  IMPORT strings

DESCRIPTION
  left returns a new String containing the first count Unicode scalar values of
  value, taken from the start of the string toward the end.

  Lengths are measured in Unicode scalar values, not UTF-8 bytes and not
  user-perceived grapheme clusters. A multi-byte character such as "é" or "😀"
  counts as one scalar even though it occupies several bytes, and left never
  splits a multi-byte scalar.

  A count of 0 returns the empty string. When count is greater than or equal to
  the scalar length of value, the entire string is returned; left does not pad
  or raise an error in that case. count must not be negative.

  left does not mutate value and has no side effects. The returned String is a
  new owned value.

PARAMETERS
  value AS String
    The string whose leading scalars are returned.

  count AS Integer
    The number of leading Unicode scalar values to take. Must be 0 or greater.
    Values at or above the scalar length of value yield the whole string.

RETURN VALUE
  AS String
    A new String holding the first count scalars of value. Returns the empty
    string when count is 0, and the whole of value when count is at least the
    scalar length of value.

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when count is negative.

TYPE CHECKING
  value must be String. count must be an Integer.

EXAMPLES
  Take the first three scalars:

    LET head AS String = strings::left("Hello", 3)

  A count beyond the length returns the whole string:

    LET whole AS String = strings::left("Hi", 10)

  Multi-byte characters count as single scalars:

    LET prefix AS String = strings::left("café", 3)
