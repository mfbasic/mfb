NAME
  writeTextAtomic - atomically replace a text file

SYNOPSIS
  fs::writeTextAtomic(path AS String, value AS String) AS Nothing

DESCRIPTION
  Writes value as UTF-8 text to a uniquely named temporary file in the same
  directory as path, flushes it to disk, and then replaces path with an OS
  rename. Readers only ever see the old or the fully written new file, never a
  partially written one. The final replacement is atomic when the host
  filesystem supports atomic rename.

ERRORS
  Raises an Error when path is empty, when the containing directory does not
  exist or is not writable, when the temporary file cannot be created, or when
  writing, flushing, closing, or renaming fails. The original path is left
  unchanged when the write fails before the final rename.

EXAMPLES
  fs::writeTextAtomic("target/output.txt", "done")

SEE ALSO
