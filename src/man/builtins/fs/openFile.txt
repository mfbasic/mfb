NAME
  openFile - open a file resource

SYNOPSIS
  fs::openFile(path AS String, mode AS String = "read") AS File

DESCRIPTION
  Opens path and returns an opaque File resource. The default mode is "read".
  Portable modes are "read" or "r", "write" or "w", "readWrite" or "rw",
  and "append" or "a".

EXAMPLES
  RES f = fs::openFile("data.txt")
  ' f is closed by lexical drop when this scope ends
