NAME
  deleteDirectory - delete an empty directory

SYNOPSIS
  fs::deleteDirectory(path AS String) AS Nothing

DESCRIPTION
  Deletes an empty directory. The final path must name an actual directory.

  Fails with ErrInvalidArgument (10002) for an empty path or embedded NUL,
  ErrNotFound (10004) when the path does not exist, ErrAccessDenied (10013)
  when deletion is denied, ErrDirectoryNotEmpty (10018) when the directory is
  not empty, or ErrOutput (10015) for other host failures.

EXAMPLES
  fs::deleteDirectory("target/example")
