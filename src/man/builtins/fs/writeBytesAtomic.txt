NAME
  writeBytesAtomic - atomically replace a binary file

SYNOPSIS
  fs::writeBytesAtomic(path AS String, bytes AS List OF Byte) AS Nothing

DESCRIPTION
  Writes bytes to a temporary path next to path and then replaces path with an
  OS rename operation. The final replacement is atomic when the host filesystem
  supports atomic rename.

EXAMPLE
  fs::writeBytesAtomic("target/output.bin", [72, 105])
