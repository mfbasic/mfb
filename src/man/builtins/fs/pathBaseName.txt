NAME
  pathBaseName - return the final path component

SYNOPSIS
  fs::pathBaseName(path AS String) AS String

DESCRIPTION
  Returns the final component of path without accessing the filesystem.

EXAMPLES
  io::print(fs::pathBaseName("target/output.txt"))

FIXME
Untested: empty path, path with no separator, multiple trailing slashes, repeated separators, `"."`, `".."`, basename after `"/a//b"`, and non-slash separators such as backslash/drive-style paths.
