NAME
  fs - Filesystem path and file-handle functions

SYNOPSIS
  IMPORT fs
  fs::exists(path)

DESCRIPTION
  The fs package provides filesystem path operations and scoped file-handle I/O.
  Path values are UTF-8 String values. File handles use the opaque File
  resource type and should be scoped with USING when possible.

  One-shot path functions operate directly on paths. File-handle functions use
  fs::open, fs::openFile, fs::openFileNoFollow, or fs::createTempFile to create
  File values that are closed with fs::close or automatically at END USING.

SEE ALSO
  errors
  types
