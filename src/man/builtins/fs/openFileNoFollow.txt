NAME
  openFileNoFollow - open a file without following the final symlink

SYNOPSIS
  fs::openFileNoFollow(path AS String, mode AS String = "read") AS File

DESCRIPTION
  Opens path like fs::openFile, but asks the host OS not to follow a symlink in
  the final path component. Portable modes are "read" or "r", "write" or "w",
  "readWrite" or "rw", and "append" or "a".

EXAMPLES
  RES f = fs::openFileNoFollow("data.txt", "read")
  ' f is closed by lexical drop when this scope ends
