NAME
  setCurrentDirectory - change the process current working directory

SYNOPSIS
  fs::setCurrentDirectory(path AS String) AS Nothing

DESCRIPTION
  Changes the current working directory for the running process.

  Fails with ErrInvalidArgument (77050002) for an empty path or embedded NUL,
  ErrNotFound (77050004) when the path does not exist, ErrAccessDenied (77030003)
  when traversal is denied, or ErrWriteFailed (77020002) for other host failures.

EXAMPLES
  fs::setCurrentDirectory("tests")
  fs::setCurrentDirectory("..")
