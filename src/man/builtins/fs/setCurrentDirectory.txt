NAME
  setCurrentDirectory - change the process current working directory

SYNOPSIS
  fs::setCurrentDirectory(path AS String) AS Nothing

DESCRIPTION
  Changes the current working directory for the running process.

  Fails with ErrInvalidArgument (10002) for an empty path or embedded NUL,
  ErrNotFound (10004) when the path does not exist, ErrAccessDenied (10013)
  when traversal is denied, or ErrOutput (10015) for other host failures.

EXAMPLES
  fs::setCurrentDirectory("tests")
  fs::setCurrentDirectory("..")
