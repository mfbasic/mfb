NAME
  fileExists - test whether a path names a regular file

SYNOPSIS
  fs::fileExists(path AS String) AS Boolean

DESCRIPTION
  Returns TRUE when path exists and resolves to a regular file. The final path
  component is followed when it is a symlink.

  Returns FALSE for missing paths, directories, and other non-regular
  filesystem entries.

  The path is a UTF-8 String and may contain Unicode characters, including
  emoji, when the host filesystem accepts those names.

EXAMPLES
  fs::fileExists("data.txt")
  fs::fileExists("é日😀.txt")

SEE ALSO
  fs::directoryExists, fs::exists
