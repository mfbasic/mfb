NAME
  createDirectory - create one directory

SYNOPSIS
  fs::createDirectory(path AS String) AS Nothing

DESCRIPTION
  Creates a single directory. The parent directory must already exist.

  Fails with ErrInvalidArgument (10002) for an empty path or embedded NUL,
  ErrAlreadyExists (10005) when the final path already exists,
  ErrAccessDenied (10013) when creation is denied, or ErrOutput (10015) for
  other host failures.

EXAMPLES
  fs::createDirectory("target/example")
