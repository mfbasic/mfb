NAME
  readAllBytes - read all remaining bytes from a File into a List OF Byte

SYNOPSIS
  fs::readAllBytes(file AS File) AS List OF Byte

PACKAGE
  fs

IMPORTS
  IMPORT fs

DESCRIPTION
  Reads every remaining byte of file, starting at the file's current read
  position and continuing to end of input, and returns them in a single
  contiguous List OF Byte collection. The file position is advanced to end of
  input, so a subsequent fs::eof reports true. file must be an open File
  resource, such as one returned by fs::openFile or fs::open, that was opened in
  a mode permitting reading.

  The amount read is determined up front: the function measures the bytes between
  the current position and the end of the file, allocates a List OF Byte of
  exactly that size, and reads the remainder into it in one or more host reads
  until the collection is full. No newline translation, decoding, or UTF-8
  validation is performed, so the returned list holds the file's remaining bytes
  exactly as stored on disk, making it suitable for binary data as well as text.
  When file is already at end of input, no bytes remain and the empty List OF
  Byte is returned.

  The function only reads from and repositions file; it does not close it and has
  no other side effects. To read the same data as validated UTF-8 text, use
  fs::readAll. To read a whole file by path in a single call rather than from an
  open handle, use fs::readBytes.

  Thread cancellation is cooperative. The current runtime does not asynchronously
  interrupt arbitrary host file reads; workers that need prompt cancellation
  around blocking file descriptors should check thread::isCancelled between
  cancellation-point operations.

PARAMETERS
  file AS File
    An open File resource to read from, positioned at the start of the data to
    read. Must not have been closed and must have been opened in a mode that
    permits reading ("read", "readWrite", or "append").

RETURN VALUE
  AS List OF Byte
    The remaining contents of the file from the current position to end of input,
    in file order, as a List OF Byte. When file is already at end of input, an
    empty List OF Byte is returned.

ERRORS
  77030004 (ErrResourceClosed)
  Raised when file has already been closed.

  77020001 (ErrRead)
  Raised when repositioning the file to measure its remaining length fails, when
  file was not opened for reading, or when the host OS read fails partway through
  before the measured bytes have all been read.

  77010001 (ErrOutOfMemory)
  Raised when the List OF Byte that holds the file's remaining contents cannot be
  allocated.

EXAMPLES
  Read all remaining bytes from an open file:

    RES f = fs::openFile("data.bin")
    LET bytes AS List OF Byte = fs::readAllBytes(f)
    ' f is closed by lexical drop when this scope ends

  Skip the first line, then read the remaining bytes of the file:

    RES f = fs::openFile("data.bin")
    LET header AS String = fs::readLine(f)
    LET body AS List OF Byte = fs::readAllBytes(f)
