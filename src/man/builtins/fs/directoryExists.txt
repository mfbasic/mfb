NAME
  directoryExists - test whether a path names a directory

SYNOPSIS
  fs::directoryExists(path AS String) AS Boolean

DESCRIPTION
  Returns TRUE when path exists and resolves to a directory. The final path
  component is followed when it is a symlink.

  Returns FALSE for missing paths, regular files, and other non-directory
  filesystem entries.

  The path is a UTF-8 String and may contain Unicode characters, including
  emoji, when the host filesystem accepts those names.

EXAMPLES
  fs::directoryExists("data")
  fs::directoryExists("é日😀")

SEE ALSO
  fs::fileExists, fs::exists
