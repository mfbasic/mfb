NAME
  getUnderline - read whether the underline text attribute is currently enabled

SYNOPSIS
  term::getUnderline() AS Boolean

PACKAGE
  term

IMPORTS
  IMPORT term

DESCRIPTION
  Returns the current state of the underline text attribute used for text drawn
  through the term:: surface, as a Boolean. The value reflects the most recent
  term::setUnderline call, or the default set by term::on if the underline
  attribute has not been changed since TUI mode was last entered.

  term::on resets the underline attribute to off, so immediately after term::on
  and before any term::setUnderline call, term::getUnderline returns False. After
  a term::setUnderline(True) call it returns True, and after term::setUnderline(False)
  it returns False, until the next term::setUnderline or the next term::on. The
  reported value tracks only the underline attribute; the foreground color,
  background color, and bold attribute are reported by their own queries and do
  not affect this value.

  Like the rest of the term:: surface other than term::on and term::isOn,
  term::getUnderline is gated behind TUI mode: it is a no-op while TUI mode is
  off, so a program must call term::on before the returned value is meaningful.
  The function takes no arguments and has no side effects: it only reads the
  current underline attribute and does not change any term:: state, the cursor,
  or the screen.

PARAMETERS
  None.

RETURN VALUE
  AS Boolean
    Returns True if the underline attribute is currently enabled for subsequently
    drawn text, or False if it is disabled. Immediately after term::on it is
    False; otherwise it is the value given to the most recent term::setUnderline
    call.

ERRORS
  No errors.

EXAMPLES
  Enable underline and read the attribute back:

    IMPORT term

    term::on()
    term::setUnderline(True)
    LET u AS Boolean = term::getUnderline()
    term::off()

  Branch on the current underline state:

    IMPORT term
    IMPORT io

    term::on()
    IF term::getUnderline() THEN
      io::print("underline is on")
    ELSE
      io::print("underline is off")
    END IF
    term::off()
