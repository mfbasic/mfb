NAME
  setBold - enable or disable the bold text attribute on the terminal surface

SYNOPSIS
  term::setBold(enabled AS Boolean) AS Nothing

PACKAGE
  term

IMPORTS
  IMPORT term

DESCRIPTION
  Turns the bold text attribute on or off for subsequent text drawn through the
  term:: surface. When enabled is True the bold attribute is switched on; when it
  is False the attribute is switched off. The change takes effect immediately and
  persists until it is changed by another term::setBold call or reset to the
  default by term::on; it does not affect text already on the screen, only text
  drawn after the call returns.

  The bold attribute is independent of the foreground color, background color, and
  underline attribute: setting it leaves those unchanged. Bold is a boolean
  attribute, so calling term::setBold(True) twice in a row has the same effect as
  calling it once.

  Like the rest of the term:: surface other than term::on and term::isOn,
  term::setBold is gated behind TUI mode: it is a no-op while TUI mode is off, so a
  program must call term::on first for the attribute change to take effect.
  term::on resets the bold attribute back to off, so any state set with
  term::setBold is discarded the next time term::on is called.

  The function has no return value; its only effect is to update the current bold
  attribute. The attribute set here can be read back with term::getBold.

PARAMETERS
  enabled AS Boolean
    Whether the bold attribute should be on. True enables bold text; False
    disables it.

RETURN VALUE
  AS Nothing
    Returns nothing. The function is invoked for its side effect of setting the
    current bold attribute used for subsequently drawn text.

ERRORS
  No errors.

EXAMPLES
  Draw bold text on the TUI surface:

    IMPORT term
    IMPORT io

    term::on()
    term::setBold(True)
    io::print("this is bold")
    term::setBold(False)
    io::print("this is not")
    term::off()

  Set the bold attribute and read it back:

    IMPORT term

    term::on()
    term::setBold(True)
    LET b AS Boolean = term::getBold()
    term::off()
