NAME
  mid - return a substring or sublist

SYNOPSIS
  mid(value AS String, start AS Integer, count AS Integer) AS String
  mid(value AS List OF T, start AS Integer, count AS Integer) AS List OF T

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  mid returns a slice of value beginning at zero-based start and containing
  count elements.

  For String values, start and count are measured in Unicode scalar values. They
  are not byte offsets and not grapheme-cluster indexes.

  For List values, start and count are measured in list items.

OVERLOADS
  mid(value AS String, start AS Integer, count AS Integer) AS String
    Returns a substring by zero-based Unicode scalar index.

  mid(value AS List OF T, start AS Integer, count AS Integer) AS List OF T
    Returns a sublist by zero-based item index.

ERRORS
  10001
    Returned when start or count describes an invalid range.

  Other errors can propagate from evaluating arguments before mid is called.

TYPE CHECKING
  mid is defined only for String and List OF T values. start and count must be
  Integer values.

EXAMPLES
  Slice text:

    LET part AS String = mid("hello", 1, 3)

  Slice a list:

    LET middle AS List OF Integer = mid([1, 2, 3, 4], 1, 2)
