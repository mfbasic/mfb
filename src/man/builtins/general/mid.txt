NAME
  mid - return a substring by Unicode scalar index

SYNOPSIS
  mid(value AS String, start AS Integer, count AS Integer) AS String

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  mid returns a slice of value beginning at zero-based start and containing
  count Unicode scalar values.

  Positions are measured in Unicode scalar values, not UTF-8 bytes and not
  user-perceived grapheme clusters. A multi-byte character such as "é" or "😀"
  counts as one position even though it occupies several bytes. The returned
  String is a new owned value.

  mid is also defined for collections; see collection mid for the List overload.

OVERLOADS
  mid(value AS String, start AS Integer, count AS Integer) AS String
    Returns a substring by zero-based Unicode scalar index.

ERRORS
  77050001
    Returned when start is less than 0, count is less than 0, or start + count
    is greater than the scalar length of value.

  Other errors can propagate from evaluating value, start, or count before mid
  is called.

TYPE CHECKING
  value must be String. start and count must be Integer values.

EXAMPLES
  Slice text:

    LET part AS String = mid("hello", 1, 3)

  Multi-byte characters count as single positions:

    LET tail AS String = mid("café", 3, 1)

SEE ALSO
  find, replace, len, collection mid
