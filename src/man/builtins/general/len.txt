NAME
  len - return the number of elements in a string, list, or map

SYNOPSIS
  len(value AS String) AS Integer
  len(value AS List OF T) AS Integer
  len(value AS Map OF K TO V) AS Integer

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  len returns the size of value as an Integer. The exact meaning of "size"
  depends on the kind of value supplied.

  For a String, len returns the number of Unicode scalar values it contains.
  The count is measured in Unicode scalar values, not UTF-8 bytes and not
  user-perceived grapheme clusters. A multi-byte character such as "é" or "😀"
  counts as one scalar value even though it occupies several bytes, while a
  grapheme made of a base character plus combining marks counts as more than one
  scalar value. Use strings::byteLen for the byte length and strings::graphemes
  when grapheme clusters are needed.

  For a List OF T, len returns the number of elements, regardless of the element
  type T. For a Map OF K TO V, len returns the number of key/value entries.

  An empty String, empty List, or empty Map returns 0. len reads value only and
  has no side effects; it never mutates its argument.

OVERLOADS
  len(value AS String) AS Integer
    Returns the number of Unicode scalar values in value.

  len(value AS List OF T) AS Integer
    Returns the number of elements in value for any element type T.

  len(value AS Map OF K TO V) AS Integer
    Returns the number of key/value entries in value for any key type K and
    value type V.

PARAMETERS
  value AS String
    The string whose Unicode scalar values are counted.

  value AS List OF T
    The list whose elements are counted, for any element type T.

  value AS Map OF K TO V
    The map whose entries are counted, for any key type K and value type V.

RETURN VALUE
  AS Integer
    The number of scalar values, list elements, or map entries in value. Returns
    0 for an empty string, list, or map.

ERRORS
  No errors.

TYPE CHECKING
  len is generic over its single argument and accepts a String, a List OF T, or
  a Map OF K TO V. Any other argument type is rejected at compile time. The
  element type T and the map key and value types K and V are unconstrained.

EXAMPLES
  Count string scalar values:

    LET count AS Integer = len("hello")

  Multi-byte characters count as single scalar values:

    LET emoji AS Integer = len("😀")

  Count the elements of a list:

    LET items AS List OF Integer = [10, 20, 30]
    LET total AS Integer = len(items)

  Count the entries of a map:

    LET ages AS Map OF String TO Integer = {"ada": 36, "bo": 9}
    LET pairs AS Integer = len(ages)

SEE ALSO
  isEmpty, isNotEmpty, strings::byteLen, strings::graphemes
