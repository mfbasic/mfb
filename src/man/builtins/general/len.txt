NAME
  len - return the number of Unicode scalar values in a string

SYNOPSIS
  len(value AS String) AS Integer

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  len returns the number of Unicode scalar values in a String.

  The count is measured in Unicode scalar values, not UTF-8 bytes and not
  user-perceived grapheme clusters. A multi-byte character such as "é" or "😀"
  counts as one scalar value even though it occupies several bytes, while a
  grapheme made of a base character plus combining marks counts as more than
  one scalar value. Use strings::byteLen for the byte length and
  strings::graphemes when grapheme clusters are needed.

  len is also defined for collections; see collection len for the List and Map
  overloads.

OVERLOADS
  len(value AS String) AS Integer
    Returns the number of Unicode scalar values in value.

ERRORS
  len does not fail for supported argument types. Errors can still propagate from
  evaluating the argument expression before len is called.

TYPE CHECKING
  General len is defined only for String. Collection overloads are documented
  under mfb man collection len.

EXAMPLES
  Count string scalar values:

    LET count AS Integer = len("hello")

  Multi-byte characters count as single scalar values:

    LET emoji AS Integer = len("😀")

SEE ALSO
  collection len, isEmpty, isNotEmpty, strings::byteLen, strings::graphemes
