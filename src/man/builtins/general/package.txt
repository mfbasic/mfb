NAME
  general - Core built-in functions

SYNOPSIS
  mfb man general
  mfb man general function

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  general contains the built-in functions that are always in scope. These
  functions cover collection and string length, numeric conversion, type
  inspection, and simple value predicates.

  Many general functions are overloaded. For example, len accepts String,
  List OF T, and Map OF K TO V values; and toString accepts only the primitive
  and byte-list overloads documented by its function page.

  List and Map helpers now live in the collections package, and string
  slicing and search helpers live in the strings package.

ERRORS
  Fallible general functions return Result and participate in normal MFBASIC
  error propagation. For example, conversions fail for invalid formats or
  out-of-range values.

EXAMPLES
  Count list items:

    LET count AS Integer = len([1, 2, 3])

  Convert a value to text:

    LET text AS String = toString(42)
