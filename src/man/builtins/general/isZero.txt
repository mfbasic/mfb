NAME
  isZero - test whether a number is zero

SYNOPSIS
  isZero(value AS Integer) AS Boolean
  isZero(value AS Float) AS Boolean
  isZero(value AS Fixed) AS Boolean

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  isZero returns TRUE when value is zero. It accepts Integer, Float, and Fixed
  values.

OVERLOADS
  isZero(value AS Integer) AS Boolean
    Returns TRUE when value = 0.

  isZero(value AS Float) AS Boolean
    Returns TRUE when value = 0.0.

  isZero(value AS Fixed) AS Boolean
    Returns TRUE when value = 0.0.

ERRORS
  isZero does not fail for supported evaluated arguments. Errors can still
  propagate from evaluating value before isZero is called.

TYPE CHECKING
  isZero accepts only Integer, Float, and Fixed values.

EXAMPLES
  Test one value:

    LET ok AS Boolean = isZero(0)

  Filter numbers:

    LET values AS List OF Integer = filter([-1, 0, 2], isZero)
