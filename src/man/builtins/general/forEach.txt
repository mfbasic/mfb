NAME
  forEach - run an action for each list item

SYNOPSIS
  forEach OF T(value AS List OF T, action AS FUNC(T) AS Nothing) AS Nothing

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  forEach calls action once for each item in value, from left to right.

  A SUB(T) is accepted for action because SUB success type is Nothing. If action
  fails for any item, the error auto-propagates and iteration stops.

OVERLOADS
  forEach OF T(value AS List OF T, action AS FUNC(T) AS Nothing) AS Nothing
    Runs action for every item.

ERRORS
  forEach propagates errors raised by action. Errors can also propagate from
  evaluating value or action before forEach is called.

TYPE CHECKING
  action must accept one argument of the list element type T and return Nothing.

EXAMPLES
  Print each string:

    forEach(["hello", "world"], io::print)

  Use a local action:

    SUB printNumber(value AS Integer)
      io::print(toString(value))
    END SUB

    forEach([1, 2, 3], printNumber)
