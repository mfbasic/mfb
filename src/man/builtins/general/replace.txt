NAME
  replace - replace matching text or list items

SYNOPSIS
  replace(value AS String, old AS String, new AS String) AS String
  replace OF T(value AS List OF T, old AS T, new AS T) AS List OF T

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  replace returns a new value with matches replaced. The original value is not
  mutated.

  For String values, replace substitutes all non-overlapping occurrences of old
  with new.

  If old is an empty String, replace returns value unchanged.

  For List values, replace substitutes every item equal to old with new. The
  element type T must be comparable.

OVERLOADS
  replace(value AS String, old AS String, new AS String) AS String
    Replaces all non-overlapping text occurrences.

  replace OF T(value AS List OF T, old AS T, new AS T) AS List OF T
    Replaces every matching list item.

ERRORS
  replace does not fail for supported argument types and valid evaluated
  arguments. Errors can still propagate from evaluating arguments before replace
  is called.

TYPE CHECKING
  String replacement requires String old and new values. List replacement
  requires old and new to have the list element type. List element type T must be
  comparable.

EXAMPLES
  Replace text:

    LET text AS String = replace("hello", "l", "x")

  Replace list items:

    LET values AS List OF Integer = replace([1, 2, 1], 1, 9)
