NAME
  isOdd - test whether an integer is odd

SYNOPSIS
  isOdd(value AS Integer) AS Boolean

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  isOdd returns TRUE when value is an odd integer and FALSE when value is even.

  An integer is odd when it is not evenly divisible by two, that is when
  value MOD 2 is not 0. Zero is even, so isOdd(0) is FALSE. Negative integers
  follow the same rule, so isOdd(-3) is TRUE and isOdd(-4) is FALSE.

  isOdd is useful in ordinary conditionals, MATCH guards, and as a predicate
  passed to filter.

PARAMETERS
  value AS Integer
    The integer to test. Any Integer is accepted; its value determines the
    result.

RETURN VALUE
  AS Boolean
    TRUE when value is not evenly divisible by two, FALSE otherwise. Zero
    returns FALSE; negative values are tested by the same parity rule as
    positive ones.

ERRORS
  No errors.

TYPE CHECKING
  isOdd accepts only Integer values. Calling it with any other type is a
  compile-time error.

EXAMPLES
  Test a literal:

    LET result AS Boolean = isOdd(7)

  Branch on parity:

    IF isOdd(count) THEN
      PRINT "count is odd"
    END IF

  Filter a list:

    LET odds AS List OF Integer = filter([1, 2, 3, 4], isOdd)
