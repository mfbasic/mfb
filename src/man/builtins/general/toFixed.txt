NAME
  toFixed - convert text, Integer, or Float to Fixed

SYNOPSIS
  toFixed(value AS String) AS Fixed
  toFixed(value AS Integer) AS Fixed
  toFixed(value AS Float) AS Fixed

PACKAGE
  general

IMPORTS
  None. general functions are always available without an IMPORT statement.

DESCRIPTION
  toFixed converts a supported value to Fixed, MFBASIC's deterministic binary
  fixed-point type. A Fixed value holds a signed 32-bit integer part and a
  32-bit fractional part, so its representable range is approximately
  -2147483648 through 2147483647 with a fractional resolution of 1/2^32.

  The String overload parses decimal fixed-point text. An optional leading
  minus sign is accepted, followed by digits and an optional fractional part;
  the entire string must form a single well-formed decimal number, so
  surrounding whitespace and other stray characters are rejected. The parsed
  value is rounded to the nearest representable Fixed value, and a magnitude too
  large for the Fixed range fails rather than wrapping.

  The Integer overload converts a signed 64-bit Integer to Fixed by placing it
  in the integer part exactly. Because the Fixed integer part is only 32 bits
  wide, an Integer outside the range -2147483648 through 2147483647 cannot be
  represented and fails.

  The Float overload converts a finite Float to the nearest representable Fixed
  value. The Float must be finite: NaN and infinity are rejected, and a finite
  value whose magnitude lies outside the Fixed range also fails.

  Because Fixed is binary fixed-point, the stored value reflects the nearest
  representable point and may differ slightly from the original decimal spelling
  used to create it. toFixed has no side effects beyond producing the result
  Fixed value.

OVERLOADS
  toFixed(value AS String) AS Fixed
    Parses decimal fixed-point text and rounds to the nearest representable
    Fixed value.

  toFixed(value AS Integer) AS Fixed
    Places a signed 64-bit Integer into the Fixed integer part exactly,
    provided it fits the 32-bit integer range.

  toFixed(value AS Float) AS Fixed
    Converts a finite Float to the nearest representable Fixed value.

PARAMETERS
  value AS String
    Text holding a decimal fixed-point number, optionally prefixed with a minus
    sign and including an optional fractional part. The whole string must be
    valid; no leading, trailing, or interior extra characters are permitted.

  value AS Integer
    A signed 64-bit value to place in the Fixed integer part. Must fall within
    the range -2147483648 through 2147483647.

  value AS Float
    A finite Float to convert to the nearest representable Fixed value. NaN and
    infinity are not accepted.

RETURN VALUE
  AS Fixed
    The Fixed equivalent of value, rounded to the nearest representable point.
    For Integer input within range the conversion is exact; for String and
    Float input the result may differ slightly from the source value when it
    cannot be represented exactly.

ERRORS
  77050003 (ErrInvalidFormat)
  Raised when a String value is not well-formed decimal fixed-point text, or
  when a Float value is NaN or infinity.

  77050010 (ErrOverflow)
  Raised when the value is outside the representable Fixed range: a String or
  Float whose magnitude is too large, or an Integer outside -2147483648 through
  2147483647.

TYPE CHECKING
  toFixed accepts only String, Integer, and Float values. Calling it with any
  other type is a compile-time error; convert such values explicitly first.

EXAMPLES
  Parse decimal text:

    LET amount AS Fixed = toFixed("2.25")

  Convert an Integer:

    LET amount AS Fixed = toFixed(10)

  Convert a Float:

    LET amount AS Fixed = toFixed(1.5)
