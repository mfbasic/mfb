NAME
  isPositive - test whether a number is greater than zero

SYNOPSIS
  isPositive(value AS Integer) AS Boolean
  isPositive(value AS Float) AS Boolean
  isPositive(value AS Fixed) AS Boolean

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  isPositive returns TRUE when value is strictly greater than zero and FALSE
  otherwise.

  The test is the strict sign test value > 0. Zero is not positive, so
  isPositive(0) is FALSE. Negative values are also not positive, so
  isPositive(-5) is FALSE, while isPositive(5) is TRUE.

  isPositive is overloaded for the three numeric types: Integer, Float, and
  Fixed. The argument is compared against zero in its own type with no
  conversion, so the same strict rule applies whether value is 7, 0.0001, or a
  Fixed amount. Negative zero, where the type permits it, is treated as zero and
  therefore yields FALSE.

  isPositive is useful in ordinary conditionals, MATCH guards, and as a
  predicate passed to filter.

OVERLOADS
  isPositive(value AS Integer) AS Boolean
    Tests an Integer against zero, returning TRUE when value > 0.

  isPositive(value AS Float) AS Boolean
    Tests a Float against zero, returning TRUE when value > 0.0.

  isPositive(value AS Fixed) AS Boolean
    Tests a Fixed against zero, returning TRUE when value is greater than zero.

PARAMETERS
  value AS Integer
    The number to test. Any Integer, Float, or Fixed is accepted; its value
    determines the result.

RETURN VALUE
  AS Boolean
    TRUE when value is strictly greater than zero, FALSE otherwise. Zero returns
    FALSE and negative values return FALSE.

ERRORS
  No errors.

TYPE CHECKING
  isPositive accepts a single Integer, Float, or Fixed value. Calling it with
  any other type, or with a different number of arguments, is a compile-time
  error.

EXAMPLES
  Test a literal:

    LET result AS Boolean = isPositive(7)

  Branch on sign:

    IF isPositive(balance) THEN
      PRINT "balance is positive"
    END IF

  Filter a list:

    LET positives AS List OF Integer = filter([-2, -1, 0, 1, 2], isPositive)
