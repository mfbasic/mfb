NAME
  isEven - test whether an integer is even

SYNOPSIS
  isEven(value AS Integer) AS Boolean

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  isEven returns TRUE when value is an even integer and FALSE when value is odd.

  An integer is even when it is evenly divisible by two, that is when
  value MOD 2 = 0. Zero is even, so isEven(0) is TRUE. Negative integers follow
  the same rule, so isEven(-4) is TRUE and isEven(-3) is FALSE.

  isEven is useful in ordinary conditionals, MATCH guards, and as a predicate
  passed to filter.

PARAMETERS
  value AS Integer
    The integer to test. Any Integer is accepted; its value determines the
    result.

RETURN VALUE
  AS Boolean
    TRUE when value is evenly divisible by two, FALSE otherwise. Zero returns
    TRUE; negative values are tested by the same parity rule as positive ones.

ERRORS
  No errors.

TYPE CHECKING
  isEven accepts only Integer values. Calling it with any other type is a
  compile-time error.

EXAMPLES
  Test a literal:

    LET result AS Boolean = isEven(4)

  Branch on parity:

    IF isEven(count) THEN
      PRINT "count is even"
    END IF

  Filter a list:

    LET evens AS List OF Integer = filter([1, 2, 3, 4], isEven)
