NAME
  find - locate text

SYNOPSIS
  find(value AS String, needle AS String, start AS Integer = 0) AS Integer

PACKAGE
  general

IMPORTS
  None. general functions are available without an IMPORT statement.

DESCRIPTION
  find returns the zero-based position of the first match at or after start.

  For String values, indexes are Unicode scalar indexes. They are not byte
  offsets and not grapheme-cluster indexes.

OVERLOADS
  find(value AS String, needle AS String, start AS Integer = 0) AS Integer
    Returns the first scalar index where needle occurs in value.

ERRORS
  ErrNotFound (10004)
    Returned when no match exists at or after start.

  10001
    Returned when start is out of range.

  Other errors can propagate from evaluating value, needle, or start before find
  is called.

TYPE CHECKING
  String search requires a String needle.

EXAMPLES
  Find text:

    LET index AS Integer = find("hello", "l")

  Start after an earlier match:

    LET index AS Integer = find("hello", "l", 3)

  Handle absence explicitly:

    IMPORT errorCode

    MATCH find("abc", "z")
      CASE Ok(i) : io::print("found at " & toString(i))
      CASE Err(e) WHEN e.code = errorCode::ErrNotFound : io::print("not found")
      CASE Err(e) : FAIL e
    END MATCH
