NAME
  dot - dot (inner) product

SYNOPSIS
  vector::dot(a AS T_N, b AS T_N) AS T

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns the dot product a.x*b.x + a.y*b.y + ..., summed in declared
  order. Exact integer arithmetic for Integer (no rounding); Float and
  Fixed use their arithmetic. The sign tells you whether the vectors
  point roughly the same way (positive) or opposite (negative).

PARAMETERS
  a AS T_N
    the first vector

  b AS T_N
    the second vector, same type as a

RETURN VALUE
  AS T
    the dot product, in the element type

ERRORS
  77050010 (ErrOverflow)
  Raised when an Integer or Fixed result exceeds the representable range,
  as in the scalar math:: functions.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Dot product of two vectors:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::dot(vector::Float3[1.0, 2.0, 3.0], vector::Float3[4.0, 5.0, 6.0])))
      RETURN 0
    END FUNC
