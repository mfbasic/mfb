NAME
  length - Euclidean length (magnitude) of a vector

SYNOPSIS
  vector::length(v AS T_N) AS T

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns the Euclidean length sqrt(x*x + y*y + ...) of v, summed in
  declared component order. For Float the square root is the hardware
  FSQRT; for Fixed it is the deterministic Q32.32 square root; for
  Integer it is a deterministic integer square root of the squared sum,
  rounded half away from zero (so it is exact for large sums with no
  floating-point round-trip). The return type is the element type T.

PARAMETERS
  v AS T_N
    the vector to measure

RETURN VALUE
  AS T
    the magnitude, in the element type

ERRORS
  None.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Length of a 3-4-5 vector:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::length(vector::Float3[3.0, 0.0, 4.0])))
      RETURN 0
    END FUNC
