NAME
  scale - component-wise (Hadamard) product

SYNOPSIS
  vector::scale(a AS T_N, b AS T_N) AS T_N

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns the component-wise product (a.x*b.x, a.y*b.y, ...). Exact for
  every element type.

PARAMETERS
  a AS T_N
    the first vector

  b AS T_N
    the second vector, same type as a

RETURN VALUE
  AS T_N
    the component-wise product

ERRORS
  77050010 (ErrOverflow)
  Raised when an Integer or Fixed result exceeds the representable range,
  as in the scalar math:: functions.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Component-wise scaling:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::scale(vector::Float2[2.0,3.0], vector::Float2[4.0,5.0])))
      RETURN 0
    END FUNC
