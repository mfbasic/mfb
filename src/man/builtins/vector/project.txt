NAME
  project - vector projection of a onto b

SYNOPSIS
  vector::project(a AS T_N, b AS T_N) AS T_N

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Returns (dot(a, b) / dot(b, b)) * b: the component of a that lies along
  b. b must be non-zero. Float and Fixed use correctly-rounded division;
  Integer rounds each component half away from zero (intentionally
  lossy).

PARAMETERS
  a AS T_N
    the vector to project

  b AS T_N
    the direction to project onto; must be non-zero

RETURN VALUE
  AS T_N
    the projection of a onto b

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when b has zero length.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Project onto the +x axis:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::project(vector::Float2[2.0,2.0], vector::Float2[1.0,0.0])))
      RETURN 0
    END FUNC
