NAME
  clamp_length - cap a vector's magnitude, preserving direction

SYNOPSIS
  vector::clamp_length(v AS T_N, max AS T) AS T_N

PACKAGE
  vector

IMPORTS
  IMPORT vector

DESCRIPTION
  Caps the magnitude of v at max, leaving its direction unchanged: if
  length(v) <= max (or v is zero) v is returned unchanged, otherwise
  each component is scaled by max/length(v). max is a scalar of the
  element type and must be non-negative. Integer scaling rounds each
  component half away from zero.

PARAMETERS
  v AS T_N
    the vector to cap

  max AS T
    the maximum magnitude; must be >= 0

RETURN VALUE
  AS T_N
    v unchanged if within max, else v scaled to length max

ERRORS
  77050002 (ErrInvalidArgument)
  Raised when max is negative.

TYPE CHECKING
  Overloaded over the nine vector record types; the two vectors of a
  binary call must be the same type. T is the element type and T_N the
  N-dimensional vector. No mixed-type or cross-dimension overload exists.

EXAMPLES
  Cap a length-5 vector at 2.5:

    IMPORT vector
    IMPORT io
    FUNC main AS Integer
      io::print(toString(vector::clamp_length(vector::Float2[3.0,4.0], 2.5)))
      RETURN 0
    END FUNC

SEE ALSO
