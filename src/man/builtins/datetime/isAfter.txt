NAME
  isAfter - compare two instants

SYNOPSIS
  datetime::isAfter(a AS Instant, b AS Instant) AS Boolean

PACKAGE
  datetime

IMPORTS
  IMPORT datetime

DESCRIPTION
  datetime::isAfter is a convenience predicate over instants, equivalent to testing
  datetime::compare(a, b). isBefore is TRUE when a precedes b, isAfter when a
  follows b, and equals when they name the same instant.
