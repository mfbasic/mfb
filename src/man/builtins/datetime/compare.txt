NAME
  compare - order two instants

SYNOPSIS
  datetime::compare(a AS Instant, b AS Instant) AS Integer

PACKAGE
  datetime

IMPORTS
  IMPORT datetime

DESCRIPTION
  datetime::compare returns -1 when a is before b, 0 when they are equal, and 1
  when a is after b. For Boolean tests use datetime::isBefore, datetime::isAfter,
  or datetime::equals. Compare DateTimes by resolving them to instants first.
