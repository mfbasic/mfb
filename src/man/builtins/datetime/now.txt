NAME
  now - the current wall-clock instant (UTC)

SYNOPSIS
  datetime::now() AS Instant

PACKAGE
  datetime

IMPORTS
  IMPORT datetime

DESCRIPTION
  datetime::now reads the host's real-time clock and returns the Instant it
  names on the UTC timeline (Unix epoch, leap-second-free). The result carries
  whole seconds since 1970-01-01T00:00:00Z and a nanosecond field in
  0 .. 999_999_999. now is the only wall-clock entry point; project the result
  through a zone with datetime::toUtc, datetime::toLocal, or datetime::inZone to
  obtain civil fields.

  Internally now reads a single nanoseconds-since-epoch reading from the OS
  intrinsic, then splits it into the seconds and nanos fields of an Instant by a
  truncating divide and remainder against 1_000_000_000. The OS reading is
  non-negative and never zero in the divisor, so the split cannot trap; the
  nanosecond remainder already falls in 0 .. 999_999_999 and needs no carry.

  now is bounded by its underlying intrinsic, which reports nanoseconds since the
  epoch and is valid through year 2262. This is a limit on now, not on Instant,
  whose seconds field spans the full Integer range.

  now is one of the few datetime functions that is not pure: two calls may
  return different instants, and a program's output depends on the host clock.
  For reproducible logic, capture one instant and derive everything else from it.
  It takes no arguments, reads host clock state only, and has no side effects.

PARAMETERS
  None.
    datetime::now takes no arguments.

RETURN VALUE
  AS Instant
    The current instant on the UTC timeline. The seconds field holds whole
    seconds since 1970-01-01T00:00:00Z and the nanos field holds the
    sub-second remainder in 0 .. 999_999_999. Two calls may return different
    instants depending on the host clock.

ERRORS
  No errors.

EXAMPLES
  Capture the current instant:

    IMPORT datetime

    LET t AS Instant = datetime::now()

  Project the current instant into the local zone to read civil fields:

    IMPORT datetime

    LET t AS Instant = datetime::now()
    LET here AS DateTime = datetime::toLocal(t)

SEE ALSO
  datetime::monotonic, datetime::inZone, datetime::toLocal, datetime::toUtc
