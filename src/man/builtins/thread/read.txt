NAME
  read - read a worker outbound message

SYNOPSIS
  thread::read(t AS Thread OF Msg TO Out) AS Msg

DESCRIPTION
  Reads the next outbound message sent by the worker with thread::emit.

  Fails with ErrNotFound when no message is available.

EXAMPLES
  LET message AS String = thread::read(t)
