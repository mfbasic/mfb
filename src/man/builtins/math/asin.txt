NAME
  asin - arc sine in radians

SYNOPSIS
  math::asin(value AS Float) AS Float
  math::asin(value AS Fixed) AS Fixed

DESCRIPTION
  Returns the arc sine of value, the angle in radians whose sine is value.
  The result lies in the range [-pi/2, pi/2]. The return type matches the
  numeric type of value, which may be a Float or a Fixed.

OVERLOADS
  math::asin(value AS Float) AS Float
    Computes the arc sine using floating-point arithmetic.

  math::asin(value AS Fixed) AS Fixed
    Computes the arc sine using deterministic fixed-point (Q32.32)
    arithmetic via asin(x) = atan2(x, sqrt(1 - x^2)), so the result is
    identical across targets. The result is rounded to the nearest Fixed
    value.

ERRORS
  The arc sine is defined only on the domain [-1.0, 1.0]. The Float
  overload fails with ErrFloatDomain (77050012) when value lies outside
  this range; the Fixed overload fails with ErrInvalidArgument (77050002)
  for the same out-of-domain input.

EXAMPLES
  LET value AS Float = math::asin(0.5)

  LET scaled AS Fixed = math::asin(1.0)

SEE ALSO
