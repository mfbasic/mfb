NAME
  asin - arc sine in radians

SYNOPSIS
  math::asin(value AS Float) AS Float
  math::asin(value AS Fixed) AS Fixed

DESCRIPTION
  Returns the arc sine of value in radians.

  The Fixed overload rounds to the nearest Fixed value. Both overloads fail
  with ErrInvalidArgument (10002) when value is outside [-1.0, 1.0].

EXAMPLES
  LET value AS Float = math::asin(0.5)
