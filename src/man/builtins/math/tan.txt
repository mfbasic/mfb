NAME
  tan - tangent of an angle in radians

SYNOPSIS
  math::tan(value AS Float) AS Float
  math::tan(value AS Fixed) AS Fixed
  math::tan(values AS List OF Float) AS List OF Float

PACKAGE
  math

IMPORTS
  IMPORT math

DESCRIPTION
  Returns the trigonometric tangent of value, where value is interpreted as a
  plane angle measured in radians. The result is the ratio of sine to cosine,
  equivalently the ratio of the side opposite the angle to the side adjacent to
  it in the corresponding right triangle. Unlike sine and cosine, tangent is
  unbounded: as the angle approaches an odd multiple of pi/2 the magnitude of
  the result grows without limit.

  The function is overloaded by the numeric type of its single argument, and the
  return type always matches that type: a Float yields a Float and a Fixed
  yields a Fixed. There is no mixed-type overload; the argument selects which
  version is called. The Fixed overload computes the result as sin(value) /
  cos(value) using deterministic fixed-point (Q32.32) arithmetic and rounds it
  to the nearest representable Fixed value.

  Tangent is undefined at odd multiples of pi/2, where cosine is zero. The Float
  overload reports this by failing with ErrFloatInf when the result would be
  infinite; the Fixed overload reports it by failing with ErrInvalidArgument
  when its computed cosine is zero. The Float overload additionally guards its
  result against a non-number outcome: a non-finite input such as a positive or
  negative infinity drives the host tangent to NaN, which is reported as
  ErrFloatNan rather than returned. The Fixed overload has no NaN failure path.
  For a very large magnitude the angle's fractional part relative to a full turn
  becomes less precise, so the result loses accuracy, but the call still
  succeeds. The function performs no mutation and has no side effects.

OVERLOADS
  math::tan(value AS Float) AS Float
    Computes the tangent of a Float angle and returns it as a Float. Fails with
    ErrFloatInf when the result is infinite at an odd multiple of pi/2, and with
    ErrFloatNan when the result is not a number, which happens when value is
    non-finite (an infinity or a NaN).

  math::tan(value AS Fixed) AS Fixed
    Computes the tangent of a Fixed angle as sin(value) / cos(value) in
    fixed-point arithmetic and returns it as a Fixed, rounding the result to the
    nearest representable Fixed value. Fails with ErrInvalidArgument when the
    computed cosine is zero.

  math::tan(values AS List OF Float) AS List OF Float
    Applies tan to each element of a numeric List using NEON SIMD and
    returns a new List of the same type and length. It raises the same
    errors as the scalar overload(s) above, but a per-element failure is
    reported once, after every element has been processed (the partially
    built result List is discarded), so the error is deterministic
    regardless of which element triggered it. Float results come from a
    hand-written NEON f64 kernel (no external math library) and are
    faithfully rounded (within 1 ULP for almost all inputs).

PARAMETERS
  value AS Float | Fixed
    The angle whose tangent is computed, expressed in radians. Any value of the
    chosen numeric type is accepted except angles at which the tangent is
    undefined (odd multiples of pi/2), which raise an error. A non-finite Float
    angle yields a NaN result and is reported as ErrFloatNan.

RETURN VALUE
  AS Float | Fixed
    The tangent of value, in the same numeric type as value. The result is
    unbounded and may have either sign. The tangent of zero is zero.

ERRORS
  77050014 (ErrFloatInf)
  Raised by the Float overload when value is at an odd multiple of pi/2, where
  cosine is zero and the resulting tangent is infinite.

  77050013 (ErrFloatNan)
  Raised by the Float overload when the computed result is not a number, which
  occurs when value is non-finite, such as the tangent of a positive or negative
  infinity. The Fixed overload never raises this error.

  77050002 (ErrInvalidArgument)
  Raised by the Fixed overload when the computed cosine of value is zero, so the
  tangent is undefined.

TYPE CHECKING
  The argument must be a single Float or Fixed value. Passing an Integer, or a
  non-numeric value such as a String, Boolean, Byte, record, union, resource,
  thread, function, or lambda, is a compile-time type error.

EXAMPLES
  Tangent of a Float angle in radians:

    IMPORT math

    LET value AS Float = math::tan(0.0)

  Tangent of a Fixed angle:

    IMPORT math

    LET slope AS Fixed = math::tan(math::pi4)
