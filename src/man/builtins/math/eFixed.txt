NAME
  eFixed - mathematical constant e as Fixed

SYNOPSIS
  math::eFixed AS Fixed

PACKAGE
  math

IMPORTS
  IMPORT math

DESCRIPTION
  Euler's number e, the base of the natural logarithm, expressed as a Fixed
  value. Its value is the closest Fixed to e, approximately 2.718281828459045.

  eFixed is a constant, not a callable function: it takes no arguments and is
  referenced by name. It evaluates to the same Fixed value on every platform,
  making it suitable for deterministic decimal arithmetic where a Float is not
  desired. Use math::e for the Float form of the same constant.

PARAMETERS
  None.
    eFixed is a constant value and accepts no arguments.

RETURN VALUE
  AS Fixed
    The closest Fixed value to e, approximately 2.718281828459045.

ERRORS
  No errors.

EXAMPLES
  Read the constant into a Fixed:

    IMPORT math

    LET value AS Fixed = math::eFixed

  Compute exponential growth with Fixed arithmetic:

    IMPORT math

    LET growth AS Fixed = principal * math::eFixed

SEE ALSO
  math::e, math::piFixed, math::ln2Fixed, math::ln10Fixed, math::twoOverPiFixed
