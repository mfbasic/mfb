NAME
  seed - reseed the calling thread's random generator

SYNOPSIS
  math::seed(value AS Integer) AS Nothing

PACKAGE
  math

IMPORTS
  IMPORT math

DESCRIPTION
  Reseeds the calling thread's PCG64 pseudo-random generator from value, so
  that the thread's subsequent math::rand draws follow a sequence determined
  solely by that seed. Seeding with a fixed value makes the sequence
  deterministic and reproducible, which is useful for tests and reproducible
  simulations; any 64-bit Integer is a valid seed, including zero and negative
  values, and equal seeds produce identical sequences on the same build.

  Each thread is seeded automatically before it runs - the main thread from the
  operating system at program startup, and a thread started with thread::start
  from the thread that spawned it - so calling math::seed is only required when
  reproducible output is wanted. The call replaces the generator state as a side
  effect and returns Nothing; it produces no draw of its own. math::seed affects
  only the thread that calls it, because each thread owns an independent
  generator and stream; reseeding one thread does not disturb any other.

PARAMETERS
  value AS Integer
    The seed for the calling thread's generator. Any Integer across the full
    signed 64-bit range is accepted; the same value always yields the same
    subsequent math::rand sequence on a given build.

RETURN VALUE
  AS Nothing
    Nothing is returned. The effect is the reseeding of the calling thread's
    generator.

ERRORS
  No errors.

EXAMPLES
  Reseed for a reproducible sequence:

    IMPORT math

    math::seed(12345)
    LET first AS Integer = math::rand(1, 100)

  Reseeding with the same value reproduces the same draws:

    IMPORT math

    math::seed(42)
    LET a AS Integer = math::rand(1, 6)
    math::seed(42)
    LET b AS Integer = math::rand(1, 6)

SEE ALSO
