NAME
  ln2 - mathematical constant ln(2) as Float

SYNOPSIS
  math::ln2 AS Float

PACKAGE
  math

IMPORTS
  IMPORT math

DESCRIPTION
  The natural logarithm of 2, as a Float value approximately equal to
  0.6931471805599453. This is the closest 64-bit IEEE 754 double-precision
  representation of ln(2); it is not the exact mathematical value, which is
  irrational and cannot be represented in finite binary.

  ln2 is a constant, not a callable function. It takes no arguments and is
  referenced by name wherever a Float expression is expected, evaluating to the
  same value on every use. It performs no computation at the call site and has
  no side effects.

  This constant is commonly used to convert between natural and base-2
  logarithms: the base-2 logarithm of a value x equals math::log(x) / math::ln2.
  For a Fixed-point representation of the same constant, use math::ln2Fixed. The
  related natural-log constant math::ln10 provides ln(10).

PARAMETERS
  None.
    math::ln2 is a constant and accepts no arguments.

RETURN VALUE
  AS Float
    The Float closest to ln(2), approximately 0.6931471805599453. The same value
    is produced on every reference.

ERRORS
  No errors.

EXAMPLES
  Reference the constant directly:

    IMPORT math

    LET value AS Float = math::ln2

  Compute a base-2 logarithm from the natural logarithm:

    IMPORT math

    LET log2 AS Float = math::log(x) / math::ln2

SEE ALSO
  math::ln2Fixed, math::ln10, math::e, math::log, math::log10
