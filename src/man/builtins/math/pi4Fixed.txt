NAME
  pi4Fixed - mathematical constant pi / 4 as Fixed

SYNOPSIS
  math::pi4Fixed AS Fixed

PACKAGE
  math

IMPORTS
  IMPORT math

DESCRIPTION
  The mathematical constant pi / 4, expressed as a Fixed value. This is an
  eighth turn, equal to 45 degrees measured in radians. Its value is the
  closest Fixed to pi / 4, approximately 0.7853981633974483.

  pi4Fixed is a constant, not a callable function: it takes no arguments and is
  referenced by name. It evaluates to the same Fixed value on every platform,
  making it suitable for deterministic decimal arithmetic where a Float is not
  desired. Use math::pi4 for the Float form of the same constant.

PARAMETERS
  None.
    pi4Fixed is a constant value and accepts no arguments.

RETURN VALUE
  AS Fixed
    The closest Fixed value to pi / 4, approximately 0.7853981633974483.

ERRORS
  No errors.

EXAMPLES
  Read the constant into a Fixed:

    IMPORT math

    LET value AS Fixed = math::pi4Fixed

  Derive the eighth turn from pi:

    IMPORT math

    LET eighthTurn AS Fixed = math::piFixed / 4.0
