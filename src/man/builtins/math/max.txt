NAME
  max - larger of two numeric values

SYNOPSIS
  math::max(a AS Integer, b AS Integer) AS Integer
  math::max(a AS Float, b AS Float) AS Float
  math::max(a AS Fixed, b AS Fixed) AS Fixed

DESCRIPTION
  Returns the larger of a and b.

  The return type matches the numeric type of the arguments. Both arguments
  must be the same numeric type: two Integers, two Floats, or two Fixeds.

ERRORS
  math::max does not fail for supported argument types. Errors can still
  propagate from evaluating the argument expressions before math::max is
  called.

TYPE CHECKING
  Both arguments must share the same numeric type. Mixing numeric types, or
  passing a non-numeric value such as a String, Boolean, Byte, record, union,
  resource, thread, function, or lambda, is a compile-time type error.

EXAMPLES
  LET value AS Integer = math::max(2, 4)

  LET upper AS Float = math::max(-3.5, 1.0)

SEE ALSO
  math::min, math::clamp, math::abs
