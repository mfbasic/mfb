NAME
  min - smaller of two numeric values

SYNOPSIS
  math::min(a AS Integer, b AS Integer) AS Integer
  math::min(a AS Float, b AS Float) AS Float
  math::min(a AS Fixed, b AS Fixed) AS Fixed

DESCRIPTION
  Returns the smaller of a and b.

  The return type matches the numeric type of the arguments. Both arguments
  must be the same numeric type: two Integers, two Floats, or two Fixeds.

ERRORS
  math::min does not fail for supported argument types. Errors can still
  propagate from evaluating the argument expressions before math::min is
  called.

TYPE CHECKING
  Both arguments must share the same numeric type. Mixing numeric types, or
  passing a non-numeric value such as a String, Boolean, Byte, record, union,
  resource, thread, function, or lambda, is a compile-time type error.

EXAMPLES
  LET value AS Integer = math::min(2, 4)

  LET lower AS Float = math::min(-3.5, 1.0)

SEE ALSO
  math::max, math::clamp, math::abs
