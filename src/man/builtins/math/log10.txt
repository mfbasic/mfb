NAME
  log10 - base-10 logarithm

SYNOPSIS
  math::log10(value AS Float) AS Float
  math::log10(value AS Fixed) AS Fixed

DESCRIPTION
  Returns the base-10 logarithm of value. The return type matches the numeric
  type of value, which may be a Float or a Fixed.

OVERLOADS
  math::log10(value AS Float) AS Float
    Computes the base-10 logarithm using floating-point arithmetic.

  math::log10(value AS Fixed) AS Fixed
    Computes the base-10 logarithm using deterministic fixed-point (Q32.32)
    arithmetic, so the result is identical across targets. The result is
    rounded to the nearest Fixed value.

ERRORS
  Both overloads fail with ErrInvalidArgument (77050002) for non-positive
  input (value less than or equal to zero).

EXAMPLES
  LET value AS Float = math::log10(100.0)

  LET scaled AS Fixed = math::log10(1000.0)

SEE ALSO
  log, exp, pow, sqrt
