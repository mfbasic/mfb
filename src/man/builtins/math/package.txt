NAME
  math - Deterministic numeric helper functions

SYNOPSIS
  IMPORT math
  math::pi
  math::abs(value)

DESCRIPTION
  The math package provides deterministic numeric constants and helper
  functions. Constants are LET values. Numeric calls are overloaded by exact
  argument type; mixed numeric arguments require an explicit conversion.

  Integer and Fixed overflow fails with ErrOverflow (10028). Invalid domains,
  such as square root of a negative value or logarithm of a non-positive
  value, fail with ErrInvalidArgument (10002).

  Float math functions return only finite values. Results that would be NaN
  fail with ErrInvalidArgument (10002). Results that would be infinity fail
  with ErrOverflow (10028).

EXAMPLES
  IMPORT math

  LET limited AS Integer = math::clamp(12, 0, 10)
  LET root AS Float = math::sqrt(9.0)
  LET angle AS Float = math::atan2(1.0, 1.0)
