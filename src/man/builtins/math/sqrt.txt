NAME
  sqrt - square root for Float and Fixed

SYNOPSIS
  math::sqrt(value AS Float) AS Float
  math::sqrt(value AS Fixed) AS Fixed

DESCRIPTION
  Returns the square root of value.

  The return type matches the numeric type of value. The single argument
  may be a Float or a Fixed. The Fixed overload rounds the result to the
  nearest Fixed value.

ERRORS
  Both overloads fail with ErrInvalidArgument (77050002) when value is
  negative, which has no real square root.

EXAMPLES
  LET value AS Float = math::sqrt(9.0)

  LET side AS Fixed = math::sqrt(2.0)

SEE ALSO
  math::pow, math::exp, math::abs
