NAME
  pi - mathematical constant pi as Float

SYNOPSIS
  math::pi AS Float

DESCRIPTION
  The mathematical constant pi (the ratio of a circle's circumference to its
  diameter) as a Float value, approximately 3.141592653589793.

EXAMPLES
  LET value AS Float = math::pi

  LET circumference AS Float = 2.0 * math::pi * radius

SEE ALSO
