NAME
  sin - sine of an angle in radians

SYNOPSIS
  math::sin(value AS Float) AS Float
  math::sin(value AS Fixed) AS Fixed

DESCRIPTION
  Returns the sine of value, where value is an angle in radians.

  The return type matches the numeric type of value. The single argument
  may be a Float or a Fixed. The Fixed overload rounds the result to the
  nearest Fixed value. Defined for all inputs, so no error is produced.

EXAMPLES
  LET value AS Float = math::sin(math::pi)

  LET wave AS Fixed = math::sin(0.0)

SEE ALSO
  math::cos, math::tan, math::asin
