NAME
  twoOverPiFixed - mathematical constant 2 / pi as Fixed

SYNOPSIS
  math::twoOverPiFixed AS Fixed

PACKAGE
  math

IMPORTS
  IMPORT math

DESCRIPTION
  The mathematical constant 2 / pi, the reciprocal of half the ratio of a
  circle's circumference to its diameter, expressed as a Fixed value. Its value
  is the closest Fixed to 2 / pi, approximately 0.6366197723675814.

  twoOverPiFixed is a constant, not a callable function: it takes no arguments
  and is referenced by name. It evaluates to the same Fixed value on every
  platform, making it suitable for deterministic decimal arithmetic where a
  Float is not desired. Use math::twoOverPi for the Float form of the same
  constant.

PARAMETERS
  None.
    twoOverPiFixed is a constant value and accepts no arguments.

RETURN VALUE
  AS Fixed
    The closest Fixed value to 2 / pi, approximately 0.6366197723675814.

ERRORS
  No errors.

EXAMPLES
  Read the constant into a Fixed:

    IMPORT math

    LET value AS Fixed = math::twoOverPiFixed

  Scale a Fixed value by 2 / pi:

    IMPORT math

    LET scaled AS Fixed = amount * math::twoOverPiFixed
