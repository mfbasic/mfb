NAME
  isNegative - test whether a number is less than zero

SYNOPSIS
  isNegative(value AS Integer) AS Boolean
  isNegative(value AS Float) AS Boolean
  isNegative(value AS Fixed) AS Boolean

PACKAGE
  filter

IMPORTS
  None. filter functions are available without an IMPORT statement.

DESCRIPTION
  isNegative returns TRUE when value is less than zero. It accepts Integer,
  Float, and Fixed values.

OVERLOADS
  isNegative(value AS Integer) AS Boolean
    Returns TRUE when value < 0.

  isNegative(value AS Float) AS Boolean
    Returns TRUE when value < 0.0.

  isNegative(value AS Fixed) AS Boolean
    Returns TRUE when value < 0.0.

ERRORS
  isNegative does not fail for supported evaluated arguments. Errors can still
  propagate from evaluating value before isNegative is called.

TYPE CHECKING
  isNegative accepts only Integer, Float, and Fixed values.

EXAMPLES
  Test one value:

    LET ok AS Boolean = isNegative(-1)

  Filter numbers:

    LET values AS List OF Integer = filter([-1, 0, 2], isNegative)
