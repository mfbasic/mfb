NAME
  isOdd - test whether an Integer is odd

SYNOPSIS
  isOdd(value AS Integer) AS Boolean

PACKAGE
  filter

IMPORTS
  None. filter functions are available without an IMPORT statement.

DESCRIPTION
  isOdd returns TRUE when value MOD 2 <> 0. It is useful in ordinary
  conditionals, MATCH guards, and as a predicate for filter.

OVERLOADS
  isOdd(value AS Integer) AS Boolean
    Returns TRUE for odd Integer values and FALSE otherwise.

ERRORS
  isOdd does not fail for an evaluated Integer argument. Errors can still
  propagate from evaluating value before isOdd is called.

TYPE CHECKING
  isOdd accepts only Integer values.

EXAMPLES
  Test one value:

    LET ok AS Boolean = isOdd(3)

  Filter a list:

    LET odds AS List OF Integer = filter([1, 2, 3, 4], isOdd)

SEE ALSO
  isEven
  filter
