NAME
  isEven - test whether an Integer is even

SYNOPSIS
  isEven(value AS Integer) AS Boolean

PACKAGE
  filter

IMPORTS
  None. filter functions are available without an IMPORT statement.

DESCRIPTION
  isEven returns TRUE when value MOD 2 = 0. It is useful in ordinary
  conditionals, MATCH guards, and as a predicate for filter.

OVERLOADS
  isEven(value AS Integer) AS Boolean
    Returns TRUE for even Integer values and FALSE otherwise.

ERRORS
  isEven does not fail for an evaluated Integer argument. Errors can still
  propagate from evaluating value before isEven is called.

TYPE CHECKING
  isEven accepts only Integer values.

EXAMPLES
  Test one value:

    LET ok AS Boolean = isEven(4)

  Filter a list:

    LET evens AS List OF Integer = filter([1, 2, 3, 4], isEven)

SEE ALSO
  isOdd
