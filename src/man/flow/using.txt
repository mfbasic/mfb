NAME
  using - scoped resource blocks

SYNOPSIS
  USING name = expression
    ...
  END USING

DESCRIPTION
  USING binds a resource for a nested block and closes it automatically at the
  end of the block.

RULES
  The bound value must be a resource type.

  The binding is local to the USING body.

EXAMPLES
  USING file = fs::open(path)
    io::print(fs::readLine(file))
  END USING
