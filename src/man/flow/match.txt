NAME
  match - value-based branching

SYNOPSIS
  MATCH value
    CASE pattern
      ...
    CASE ELSE
      ...
  END MATCH

DESCRIPTION
  MATCH selects the first matching CASE branch.

RULES
  CASE patterns must be compatible with the matched value.

  CASE ELSE is optional but is commonly used as a fallback.

EXAMPLES
  MATCH result
    CASE Ok(value)
      io::print(toString(value))
    CASE ELSE
      io::print("fallback")
  END MATCH

SEE ALSO
  flow if
  errors
