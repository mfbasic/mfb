NAME
  flow - control-flow statements and block forms

SYNOPSIS
  mfb man flow

DESCRIPTION
  MFBASIC control flow is statement-based. Conditions must be Boolean. Blocks
  either use dedicated terminators such as END IF, WEND, and LOOP UNTIL, or a
  single-line form separated with `:`.

TOPICS
  if
    Conditional execution with THEN, ELSEIF, ELSE, and END IF.

  for
    Counted loops with TO, optional STEP, and NEXT.

  forEach
    Collection iteration over List and Map values.

  while
    Pre-test loops with WHILE and WEND.

  do
    Post-test and pre-test DO loop forms.

  match
    Value-based branching with CASE and END MATCH.

  using
    Resource-scoped blocks with automatic close behavior.

SEE ALSO
  types logical
  types comparisons
  errors
