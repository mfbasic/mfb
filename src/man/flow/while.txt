NAME
  while - pre-test loops

SYNOPSIS
  WHILE condition : ... : WEND
  WHILE condition
    ...
  WEND

DESCRIPTION
  WHILE evaluates its Boolean condition before each iteration.

RULES
  The condition must be Boolean.

  When the condition is FALSE before the first check, the body does not run.

EXAMPLES
  WHILE x < 10 : x = x + 1 : WEND

SEE ALSO
  flow do
  flow if
