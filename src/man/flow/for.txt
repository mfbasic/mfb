NAME
  for - counted loops

SYNOPSIS
  FOR name = start TO end : ... : NEXT
  FOR name = start TO end STEP step
    ...
  NEXT

DESCRIPTION
  FOR iterates a numeric loop variable from start toward end.

RULES
  Start, end, and step must be numeric.

  STEP is optional. The default STEP is 1.

  Positive STEP continues while the loop variable is <= end.

  Negative STEP continues while the loop variable is >= end.

  STEP must not be zero.

EXAMPLES
  FOR i = 1 TO 5 STEP 2 : io::print(toString(i)) : NEXT
  FOR down = 5 TO 1 STEP -2 : io::print(toString(down)) : NEXT
