mfb-repo

A package registry for the MFBASIC language. Every package, every version, and every state change is public and permanently visible — including versions that have been deprecated or yanked. Nothing here can log in, and nothing here can be changed from a browser.

Using the registry

Search for a package to see its versions, native target matrix, and publish history. Each package also exposes a transparency Audit tab: an append-only log with inclusion proofs, release-state transitions, and the registry’s identity-key rotation chain, so an independent monitor can check whether this registry is showing the same history to everyone.

There are no accounts and no sign-in. Every action is a plain link or a GET request; nothing on this site mutates state.
